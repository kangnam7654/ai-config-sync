## Key Benchmarks

- **CAC**: B2B SaaS average $702, DevTools (organic-heavy) $30-80, median $45
- **LTV/CAC**: Healthy >= 3:1, Strong >= 5:1; B2B SaaS target 4:1
- **CAC Payback**: Median 6.8mo across SaaS; B2B 8.6mo; healthy < 12mo
- **Freemium Conversion**: DevTools 1-3%, B2B SMB 2-5%, top quartile 8-15%
- **Monthly Churn**: Good < 1%/mo (< 5% annual); B2B SaaS avg 4.9% annual
- **Gross Margin**: SaaS 70%+, DevTools infra-light 80-90%
- **Open Source SaaS Conversion**: Typically 0.5-3%, lower than standard freemium

Source: Compiled from multiple industry reports (First Page Sage, ProductLed, OpenView, Vena) March 2026.
