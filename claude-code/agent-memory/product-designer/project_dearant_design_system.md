---
name: Dear,ANT Design System
description: Design tokens, color palette, typography, and accessibility rules for the Dear,ANT investment emotion analysis app
type: project
---

Dear,ANT uses a green-based palette (green-900 #1B4332 through green-50 #F0FFF4) with Pretendard Variable font. Mobile-first (390x844).

**Why:** The brand targets Korean 20-40s individual investors with emotion/psychology coaching for investment decisions. Must comply with capital market law (no stock recommendations).

**How to apply:**
- Text on white MUST use green-800 (#2D6A4F) minimum for WCAG AA 4.5:1 contrast. NEVER use green-500/600/700 for text on white backgrounds.
- Emotion spectrum: calm=mint-400, confident=blue-400, excited=amber-400, anxious=orange-500, fearful=red-500
- Grade colors: S=gold, A=emerald, B=blue, C=amber, D=orange, F=red (always pair color + text label for color-blind users)
- Cards: white bg, 16-20px border-radius, subtle shadow, 12-16px gap in bento grids
- Bottom nav: glassmorphism with backdrop-blur
- Accent text: use gradient clip (green-600 to green-400) for decorative emphasis rather than flat green-600 text
