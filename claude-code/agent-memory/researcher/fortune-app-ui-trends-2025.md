# Fortune/Astrology App UI Design Trends (2025-2026)

Researched: 2026-03-20. Standard depth. 16 sources consulted.

## Benchmark App Profiles

### Co-Star
- Strict monochrome: pure black + pure white, no color accents
- Typography: small-scale system font, below-average font sizes (accessibility critique)
- Minimalist anti-design: no gradients, no illustrations, no color
- Loading animation: galaxy-themed ("THE VOID" feature)
- Distinctive: abstract icons deliberately ambiguous = mystique but hurts discoverability
- Critique: feature discoverability weak; progressive disclosure via subtle CTAs

### CHANI
- Off-white backgrounds (#F5F0EB range) + blush and lavender accents
- Collage/zine illustration style: star motifs, handcrafted feel
- Typography: expressive type for titles, legible body, concise hierarchy
- Light mode primary; "whimsical, zine-like feel"
- Trauma-informed voice = gentle, ritual-like interaction design

### The Pattern
- Dark mode primary: deep navy/dark gray backgrounds, blue + teal accents
- Card-based insight layout: "Trusting Yourself", "Fears or Hesitation" cards
- Layered architecture: card summary → expand → audio series
- Animated pie charts for relationship dynamics ("Bonds" section)
- Clinical/therapy aesthetic: serious, not mystical

### 포스텔러 (ForceTeller)
- Card-based layouts, graph visualizations for fortune data
- Free-to-paid progressive disclosure model
- Lightweight presentation over text-heavy traditional formats
- Entry barrier reduction as core design principle

### 점신 (JeomShin)
- Trust-based information architecture: practitioner reviews + ratings + availability
- Interactive elements: tarot card draw, fortune cookie break, crystal ball touch
- 400+ specialists = social proof emphasis over visual flourish
- Browsing-oriented layout for practitioner selection

## Color Palette Trends (2025-2026)

### Dominant Approaches
1. Dark cosmic: deep navy (#0A0A1A), deep purple (#1A0E2E–#2D1B4E), near-black backgrounds
2. Soft mystical light: off-white (#F5F0EB), blush, lavender — used by CHANI
3. Monochrome contrast: pure black/white — Co-Star's signature

### Accent Colors (observed across design kits)
- Purple range: #7C3AED, #A855F7, #B125C0
- Blue/teal: #0057FF, #0A7494, teal accents
- Dark background soft grays: #191919, #242424, #1b1b1b (avoid pure #000000 — too harsh on OLED)

### Emerging 2026 Trends
- Glassmorphism with vibrant ambient orbs: deep purples, neon blues, hot pinks behind frosted glass
- Aurora gradient: multi-color mesh gradients mimicking Northern Lights
- Jewel tones for luxury/spiritual apps

## Typography Trends (2025-2026)

- Expressive title typography: large, bold, emotionally evocative headlines
- Acumin Pro, Poppins used in design kits (sans-serif, multiple weights)
- Serif revival for mystical/spiritual feel (editorial trend; experimental)
- Body text: generous line-height (1.4–1.45em), 15–16pt minimum
- "Variable typography" as 2026 trend: fonts that respond to context/emotion
- Co-Star critique: font sizes below average = accessibility concern

## Layout Patterns

- Card-based primary: digestible insight cards, 20pt corners, shadow depth
- Bento grid: modular tile layouts for dashboard-style home screens (2025-2026 trend)
- Progressive disclosure: summary card → expand → detail → action
- Parallax scrolling: depth and cinematic immersion
- Carousel/horizontal scroll: featured content, compatibility reports
- Bottom tab navigation: 4–5 tabs standard
- Onboarding: 8–12 steps for birth data + personalization setup

## Animation & Micro-interaction Trends

- "Light but pleasant feedback" preference over heavy motion (2026 Korean trend analysis)
- Galaxy/celestial loading animations (Co-Star's THE VOID as benchmark)
- Animated charts for relationship dynamics
- Haptic feedback integration
- Interactive divination: direct manipulation (card draw, crystal ball touch)
- Micro-interactions on CTA buttons, state transitions
- Parallax on scroll for depth

## Dark Mode vs Light Mode

- 82% of smartphone users use dark mode (2024 Nielsen data)
- Android: 81.9% dark mode adoption; iOS: 55–70%
- 82.7% switch to dark mode after 10 PM
- Fortune/astrology apps: overwhelming dark mode prevalence (Co-Star, The Pattern, Nebula, 포스텔러 all dark-primary)
- Exception: CHANI uses light-primary with soft off-white
- Best practice: avoid pure #000000 background; use #191919 or #1A0E2E for depth with OLED battery savings up to 47%

## Illustration Style Trends

1. Celestial/cosmic minimal: line-art stars, moon phases, constellation glyphs (Co-Star style)
2. Collage/zine: handcrafted, editorial, layered imagery (CHANI style)
3. Modern 3D: volumetric orbs, crystal balls, floating elements
4. Flat mystical: simplified zodiac symbols, clean vector celestials
5. Photography + overlay: real celestial photography with gradient overlays

## Accessibility Standards (WCAG 2.2, 2024 update)

- Touch targets: minimum 24×24 CSS pixels (WCAG 2.5.8); Apple HIG recommends 44×44pt
- Contrast ratio: 4.5:1 for normal text (<18pt or <14pt bold); 3:1 for large text
- Focus indicators: visible and not obscured (WCAG 2.4.11/2.4.12)
- Text scaling: honor system font size preferences
- Co-Star cited as accessibility risk: below-average font sizes throughout
- WCAG 2.2 published Nov 2023; updated May 2025. Now covers mobile apps, native apps, hybrid apps.
