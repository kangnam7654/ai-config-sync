# Comparison Reference

## When to Use
1:1 or 1:N comparison requests between technologies, tools, frameworks, or services.

## Output Format
| Criterion | {Option A} | {Option B} |
|---|---|---|
| {criterion 1} | {detail} | {detail} |
| ... | ... | ... |

## Required Sections
1. Executive Summary (1-2 sentences picking the winner with justification)
2. Comparison Table (all relevant criteria)
3. Use Case Recommendation (when to choose which)
4. Sources (URLs for all factual claims)
