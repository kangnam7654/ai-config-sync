# UX-UI Spec: TodoFlow

## 페르소나

- **이름**: 김개발 (28세, 주니어 웹 개발자)
- **목표**: 일일 할일 빠른 등록/추적, 완료 항목 시각적 구분
- **페인포인트**: 복잡한 UI, 등록에 여러 단계 필요
- **기술 수준**: 중급

## 유저 플로우

### 할일 추가
1. 입력 필드에 제목 입력
2. Enter 또는 추가 버튼 클릭
3. 목록 하단에 추가, 입력 필드 초기화

### 할일 완료 토글
1. 체크박스 클릭
2. 완료: 취소선 + 색상 변경 (#9CA3AF)
3. 재클릭으로 미완료 복원

### 할일 삭제
1. 삭제 버튼(x) 클릭
2. 즉시 삭제 (확인 없음)

## 정보 구조

단일 페이지: 앱 헤더 > 할일 입력 > 할일 목록 > (빈 상태 메시지)
