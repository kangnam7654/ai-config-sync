## 트리거 불일치 사유

1. **반복/연속 의도 부재**: 프롬프트는 "좀 개선해줘"로, 1회성 개선 요청이다. 목표 점수까지 반복적으로 돌리겠다는 의도가 전혀 표현되지 않았다.

2. **SKILL.md의 명시적 구분 규칙 위반**: "사용자가 '개선해줘'라고 해도 '반복', '계속', '점진적', '루프' 같은 키워드가 있으면 이 스킬을 사용할 것" — 역으로, 이 키워드들이 없으면 auto-improve-loop가 아닌 auto-improve(1회성)를 사용해야 한다.

3. **스킬의 자기 정의와 불일치**: auto-improve-loop는 "auto-improve(1회성 개선)와 구분: 이 스킬은 목표 도달까지 여러 라운드를 자율적으로 반복한다"라고 명시하고 있다. "좀 개선해줘"는 1회성 개선 의도이므로 auto-improve의 영역이다.
