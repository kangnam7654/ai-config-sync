# Phase 3: 수익 모델 설계

## 1. 선택한 수익 모델: 하이브리드 (포인트 IAP + 보상형 광고 + 구독)

### 선택 이유

| 근거 | 설명 |
|------|------|
| **제품 특성** | 사주/운세 앱은 사용 빈도가 불규칙 (매일 가벼운 확인 + 비정기 깊은 분석). 구독만으로는 비정기 사용자 이탈, 종량제만으로는 습관 유저 수익화 실패 |
| **경쟁사 벤치마크** | 포스텔러(코인+구독), 점신(광고+인앱). 상위 앱 모두 하이브리드 |
| **비용 구조** | 규칙 기반 운세는 비용 $0, AI 상담만 건당 50원. 무료 제공 부담이 낮아 프리미엄 전환에 집중 가능 |
| **500명 규모** | 초기 사용자 적어 광고 수익만으로 부족. 포인트 IAP로 즉시 매출, 광고로 무료 유저 수익화 보완 |
| **기존 구현** | 포인트 시스템, IAP, 광고 보상이 이미 구축되어 있어 추가 개발 비용 최소 |

### 수익원 비중 목표 (12개월 후)

| 수익원 | 비중 | 역할 |
|--------|------|------|
| 포인트팩 IAP | 50% | 핵심 매출 (비정기 구매자) |
| 구독 | 25% | 반복 매출 (MRR, 충성 유저) |
| 보상형 광고 | 15% | 무료 유저 수익화 + 리텐션 |
| 단품 리포트 (시즌) | 10% | 시즌 매출 (신년운세, 이벤트) |

## 2. 기각한 대안 모델

| 대안 | 기각 이유 |
|------|----------|
| **구독 전용** | 비정기 사용자가 많은 운세 앱에서 구독만 강제하면 이탈 큼. 포스텔러도 구독 가격 인상 후 전환율 하락 사례. 500명 규모에서 구독 MRR만으로 서버비 커버 어려움 |
| **광고 전용** | 점신 수준(DAU 수십만)이 아니면 광고만으로 수익 불가. 500명 DAU 기준 AdMob eCPM 1,000~3,000원 = 일 500~1,500원 수준. 서버비(월 5만원)도 못 커버 |
| **종량제 (건당 과금)** | 포인트 시스템과 유사하나, 직접 원화 결제는 심리적 저항 높음. 포인트/코인이 지불 고통을 완화하는 효과 입증됨 |
| **전문 상담사 수수료** | 향후 확장 계획이나, 현재 AI 전용이므로 적용 불가. 양면 시장 구축 난이도 높음 |

## 3. 가치 지표와 과금 단위

### 가치 지표 (고객이 "이것 때문에 돈을 낸다"고 느끼는 단위)

| 가치 지표 | 과금 단위 | 납득도 |
|----------|----------|--------|
| **개인화된 운세 해석의 깊이** | 포인트 (운세 유형별 차등) | 높음 - "더 자세히 보려면" |
| **AI 상담 질문 횟수** | 포인트 (질문당 10P) | 높음 - "질문할수록 비용" |
| **매일 제공되는 콘텐츠** | 구독 (월정액) | 중간 - 습관 유저 한정 |
| **광고 없는 경험** | 구독 (광고 제거) | 중간 - 광고 거부감 있는 유저 |

### 과금 단위: 포인트 (P)

- 1P 약 39원 (100P = 3,900원 기준)
- 대량 구매 시 할인: 300P = 8,900원(~30원/P), 700P = 19,900원(~28원/P)
- 포인트 소진 → 충전 유도 → 반복 구매 루프

## 5. 초기 가격 범위 (경쟁사 벤치마크 기반)

### 포지셔닝 맵

```
가격 ↑
     │  정통만세력 (전문가용, 고가)
     │
     │  포스텔러 (다양한 콘텐츠, 중~고가)
     │  신한운세 (전통 전문성, 중가)
     │
     │  ★ 달결 (AI 차별화, 저~중가)  ← 진입 포지션
     │
     │  운세의 신 (캐주얼, 저가)
     │  점신 (광고 기반, 무료~저가)
가격 ↓
```

### 가격 원칙

1. **시장 대비 30~50% 저렴하게 진입** - 500명 규모에서 전환율이 가격보다 중요
2. **3,900원 진입 장벽** - 커피 한 잔 가격, 한국 앱 시장 최저 결제 심리선
3. **포인트 시스템으로 지불 고통 완화** - 직접 원화 대비 -20% 체감 저항
4. **데이터 기반 단계적 인상** - 전환율/LTV 데이터 축적 후 가격 최적화
