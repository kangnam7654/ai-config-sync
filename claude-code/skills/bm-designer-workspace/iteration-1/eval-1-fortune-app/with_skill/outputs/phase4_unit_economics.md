## 2. 핵심 지표 산출

### 2.1 CAC (고객 획득 비용)

| 시나리오 | CAC | 산출 근거 |
|---------|-----|----------|
| 보수적 | 8,000원 | 유료 광고 의존 시 (인스타/카카오 광고) |
| 기본 | 3,000원 | ASO + SNS 바이럴 + 소규모 광고 |
| 낙관적 | 1,000원 | 순수 오가닉 + 바이럴 (궁합 공유 등) |

> **초기 전략**: 마케팅 예산 0원. ASO + 궁합 공유 바이럴로 CAC 최소화. 현실적 CAC = 0~1,000원.

### 2.2 LTV (고객 생애 가치)

```
LTV = ARPU x 평균 이용 기간 (월)

- ARPU = (유료 비율 x ARPPU) + (무료 비율 x 광고 수익/인)
- 유료 비율 = 6% (기본)
- ARPPU = 10,000원/월
- 무료 ARPU (광고) = 약 200원/월 (DAU 기준 광고 노출 수 x eCPM)
- ARPU = 0.06 x 10,000 + 0.94 x 200 = 600 + 188 = 788원/월
- 평균 이용 기간 = 1 / 월간 Churn = 1 / 0.18 = 5.6개월

LTV = 788 x 5.6 = ~4,413원 (기본 시나리오)
```

| 시나리오 | 전환율 | ARPPU | Churn | 이용기간 | ARPU/월 | LTV |
|---------|--------|-------|-------|---------|---------|-----|
| 보수적 | 3% | 8,000원 | 25% | 4개월 | 428원 | 1,712원 |
| 기본 | 6% | 10,000원 | 18% | 5.6개월 | 788원 | 4,413원 |
| 낙관적 | 10% | 12,000원 | 10% | 10개월 | 1,388원 | 13,880원 |

### 2.3 LTV/CAC 비율

| 시나리오 | LTV | CAC | LTV/CAC | 판정 |
|---------|-----|-----|---------|------|
| 보수적 | 1,712원 | 8,000원 | 0.2:1 | 위험 (유료 마케팅 불가) |
| 기본 (오가닉) | 4,413원 | 1,000원 | 4.4:1 | 건강 |
| 낙관적 (오가닉) | 13,880원 | 1,000원 | 13.9:1 | 매우 건강 |

> **핵심 인사이트**: 부트스트랩 상태에서 유료 마케팅은 비효율적. 오가닉/바이럴 성장이 필수.

### 2.4 Payback Period

### 2.5 Gross Margin

```
매출 1건 (포인트팩 100P = 3,900원 기준):
- Apple 수수료: 3,900 x 15% = 585원
- 100P 사용 시 AI API 비용: 약 10회 질문 x 50원 = 500원 (최대)
  (실제로는 규칙 기반 운세 비용 $0이 대부분)
- 평균 변동비: 약 685원 (AI 많이 쓰는 경우)
- 평균 변동비: 약 585원 (규칙 기반 운세만 쓰는 경우)

Gross Margin = (3,900 - 685) / 3,900 = 82.4% (AI 혼합)
Gross Margin = (3,900 - 585) / 3,900 = 85.0% (규칙 기반)
```

| 수익원 | 매출 | 변동비 | Gross Margin |
|--------|------|--------|-------------|
| 포인트팩 (규칙 기반 운세) | 3,900원 | 585원(Apple) | 85.0% |
| 포인트팩 (AI 상담 포함) | 3,900원 | 685원(Apple+API) | 82.4% |
| 구독 (라이트) | 6,900원 | 1,035원(Apple) | 85.0% |
| 보상형 광고 (1회) | ~1.5원 (eCPM 기반) | 0원 | 100% |

> **Gross Margin 82~85%**: 앱 비즈니스 건강 기준(60%+) 크게 초과. 규칙 기반 사주 엔진의 비용 $0이 핵심 경쟁력.

### 2.6 Contribution Margin

```
월간 기준 (기본 시나리오):
- 매출: 500명 x 6% 전환 x 10,000원 ARPPU = 300,000원
- 광고 수익: 60 DAU x 2회 광고/일 x 30일 x 1.5원 = 5,400원
- 총 매출: ~305,400원
- 변동비 (Apple 15%): 300,000 x 0.15 = 45,000원
- AI API: 약 300건 x 50원 = 15,000원
- 고정비: 65,000원
- Contribution Margin: (305,400 - 45,000 - 15,000 - 65,000) / 305,400 = 59.1%
```

### 2.7 Break-even Point

```
월 고정비: 65,000원
건당 공헌이익 (포인트팩 100P):
  3,900 - 585(Apple) - 100(평균 API) = 3,215원

BEP = 65,000 / 3,215 = 약 21건/월 (포인트팩 100P 기준)
     = 약 81,900원 매출/월

현재 500명 중 6% = 30명 유료, 평균 ARPPU 10,000원
→ 월 매출 300,000원 → BEP 초과, 흑자 가능
```

> 전환율 1%p 변동이 매출을 ~16.7% 변화시킴. **최우선 최적화 대상**.

#### 4.2 ARPPU (두 번째 민감)

| ARPPU 변동 | 월 12 매출 (기본 시나리오) | 변화율 |
|------------|-------------------------|--------|

| 12,000원 (+20%) | 693,132원 | +18.9% |

> ARPPU와 매출은 거의 1:1 비례. **고가 상품(종합 사주 리포트, AI 리포트) 개발이 ARPPU 향상의 열쇠**.

| 성장률 변동 | 월 12 누적 유저 | 월 12 매출 | 변화율 |

> 성장률은 복리 효과로 장기적으로 가장 큰 레버. **바이럴 계수(궁합 공유, 운세 카드 공유)가 핵심**.

### 민감도 요약 매트릭스

```
영향도 ↑
     │  ① 전환율 (가장 민감, 즉각 효과)
     │  ② ARPPU (비례 효과)
     │  ③ 성장률 (복리 효과, 장기적으로 가장 큼)
     │
     │  ④ Churn (간접적, 이용 기간에 영향)
     │  ⑤ AI API 비용 (낮은 영향, 원가 비중 작음)
영향도 ↓
```
