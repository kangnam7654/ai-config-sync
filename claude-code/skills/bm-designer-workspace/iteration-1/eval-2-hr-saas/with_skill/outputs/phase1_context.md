# Phase 1: 컨텍스트 수집 결과

## 수집된 정보

| 항목 | 내용 |
|------|------|
| 제품/서비스 | B2B HR SaaS 솔루션 (인사관리 종합 플랫폼) |
| 타겟 고객 | 한국 중소·중견기업 (직원 수 기반 과금 고려) |
| 현재 수익 상태 | Pre-revenue (시드 단계) |
| 경쟁 환경 | Flex, 그리팅(Greating), 원티드스페이스 |
| 비용 구조 | 시드 투자 3억 KRW, 12개월 런웨이 유지 필요 → 월 소진 ~2,500만원 이내 |
| 제약 조건 | 12개월 런웨이 제약, 시드 스테이지 리소스 한계 |

## 합리적 가정 (미확인 항목)

- HR 솔루션 범위: 근태관리, 급여, 인사정보 관리, 휴가 관리 등 핵심 HR 기능 포함
- 초기 팀 규모: 5~8명 (개발자 3~4, 비즈니스 2~3, 운영 1~2)
- 주요 비용: 인건비(~70%), 서버/인프라(~15%), 마케팅(~10%), 기타(~5%)
- 월 Burn Rate: ~2,500만원 (3억 ÷ 12개월)
- 차별점: 후발 주자로서 UX 개선, 특정 산업/규모 특화, 가격 경쟁력 확보 필요
