# B2B HR SaaS 비즈니스 모델 플랜

## 1. 요약 (Executive Summary)

직원 수 기반 per-seat 과금 모델의 HR SaaS 사업. 시드 투자 3억 원, 12개월 런웨이 확보를 전제로, 초기 시장 진입 → PMF 검증 → 매출 확대의 3단계 전략을 수립한다. 핵심 경쟁사(flex, 그리팅, 원티드스페이스)와의 차별화 포인트를 명확히 하고, 런웨이 내 BEP 도달 경로를 제시한다.

## 2. 시장 분석

### 2.1 국내 HR SaaS 시장 현황

| 지표 | 수치 |
|------|------|
| 국내 HR Tech 시장 규모 (2025 추정) | 약 1.2~1.5조 원 |
| 연평균 성장률 (CAGR) | 15~20% |
| SaaS 전환율 (온프레미스→클라우드) | 가속 중, 중소기업 중심 |
| 주요 트리거 | 재택근무 확산, MZ세대 HR 니즈, 노동법 개정 빈도 증가 |

### 2.2 경쟁사 분석

| 항목 | flex | 그리팅 | 원티드스페이스 |
|------|------|--------|----------------|
| **핵심 포지션** | 올인원 HR (근태+급여+조직) | ATS (채용 특화) | 채용+HR 통합 |
| **과금 모델** | 직원 수 기반 월정액 | 채용 건별 + 월정액 | 직원 수 기반 월정액 |
| **가격대 (인당/월)** | 약 4,000~8,000원 | 월 고정 + 건별 | 약 3,000~6,000원 |
| **무료 플랜** | 소규모 무료 (제한적) | 무료 체험 | 무료 체험 |
| **최소 계약** | 없음 (월 단위) | 없음 | 없음 |
| **타겟 기업 규모** | 10~500인 | 50~1,000인 | 50~500인 |
| **강점** | UX/UI 우수, 인지도 | 채용 프로세스 깊이 | 원티드 채용 DB 연동 |
| **약점** | 가격 부담 (소기업), 커스터마이징 한계 | HR 전반 기능 부족 | HR 기능 아직 얕음 |
| **추정 고객 수** | 10,000+ | 3,000+ | 5,000+ |
| **추정 ARR** | 200억+ | 100억+ | 80억+ |

### 2.3 시장 기회 (Gap 분석)

경쟁사 분석에서 도출된 미충족 니즈:

1. **가격 접근성**: flex 대비 30~50% 저렴한 가격대에서 핵심 기능 제공 가능
2. **기능 깊이와 폭의 균형**: 그리팅(채용만)과 flex(올인원) 사이의 중간 포지션
3. **중소기업 특화**: 10~100인 규모 기업의 HR 자동화 니즈가 가장 높으나, 기존 솔루션은 이 구간에서 가성비가 낮음
4. **온보딩/오프보딩 자동화**: 대부분의 경쟁사에서 약한 영역
5. **노동법 자동 반영**: 잦은 법 개정에 대한 자동 업데이트 니즈

## 3. 가격 전략 (Pricing Strategy)

### 3.1 과금 모델: 직원 수 기반 티어드 프라이싱

**핵심 원칙:**
- 직원 수(headcount) 기반 월 정액 과금
- 연간 결제 시 20% 할인 (현금흐름 안정화)
- 무료 티어로 초기 진입 장벽 제거
- 기능 게이팅으로 업셀 유도

### 3.2 가격표

| 플랜 | 가격 (인당/월) | 대상 기업 | 핵심 기능 | 제한 |
|------|---------------|-----------|-----------|------|
| **Free** | 0원 | 1~5인 | 근태 관리, 직원 DB, 기본 리포트 | 5인까지, 데이터 6개월 보관 |
| **Starter** | 2,500원 | 6~30인 | Free + 휴가 관리, 전자 계약, 조직도 | 관리자 2명, 기본 지원 |
| **Growth** | 4,500원 | 31~100인 | Starter + 급여 연동, 온보딩 자동화, 인사 평가, API 연동 | 관리자 5명, 우선 지원 |
| **Enterprise** | 협의 (3,500~6,000원) | 100인+ | Growth + SSO, 커스텀 워크플로우, 전담 CSM, SLA 보장 | 무제한 관리자 |

**가격 설정 근거:**
- Starter 2,500원: flex 대비 약 40% 저렴. 소기업 의사결정자의 심리적 저항선(월 10만 원) 이하로 30인까지 진입 가능
- Growth 4,500원: 경쟁사 평균 가격대와 유사하나, 기능 대비 가성비 우위
- 연간 계약 할인 후 실질 가격: Starter 2,000원, Growth 3,600원

### 3.3 가격 심리학 적용

| 전략 | 적용 방식 |
|------|-----------|
| **앵커링** | 웹사이트에서 Enterprise를 먼저 노출, Growth가 합리적으로 보이게 |
| **디코이 효과** | Growth 플랜이 Starter 대비 80% 비싸지만 기능은 3배 → Growth로 유도 |
| **무료→유료 전환** | Free 5인 제한으로 성장하는 스타트업의 자연스러운 전환 유도 |
| **연간 결제 인센티브** | 20% 할인 + 1개월 무료 온보딩 컨설팅 |

### 3.4 할인 정책

| 할인 유형 | 조건 | 할인율 |
|-----------|------|--------|
| 연간 결제 | 12개월 선결제 | 20% |
| 스타트업 | 투자 유치 2년 이내 증빙 | 30% (12개월 한정) |
| 비영리 | 비영리법인 증빙 | 50% |
| 추천 | 기존 고객 추천으로 가입 | 양측 1개월 무료 |
| 볼륨 | 200인 이상 | 별도 협의 |

## 4. 수익 모델 상세

### 4.1 수익원 구조

```
총 매출
├── [70%] 구독 매출 (SaaS MRR)
│   ├── Starter 구독
│   ├── Growth 구독
│   └── Enterprise 구독
├── [15%] 부가 서비스 매출
│   ├── 급여 아웃소싱 연동 수수료
│   ├── 전자계약 추가 건수 (월 50건 초과분)
│   └── 프리미엄 리포트/분석
├── [10%] 온보딩/컨설팅 매출
│   ├── 초기 셋업 컨설팅 (1회성)
│   └── HR 제도 설계 컨설팅
└── [5%] 파트너/연동 수수료
    ├── 급여 솔루션 연동 수수료
    └── 보험/복지몰 제휴 수수료
```

### 4.2 핵심 SaaS 지표 목표

| 지표 | 6개월 목표 | 12개월 목표 | 업계 벤치마크 |
|------|-----------|------------|--------------|
| MRR | 800만 원 | 3,000만 원 | - |
| 유료 고객 수 | 30개사 | 80개사 | - |
| ARPU (인당 평균) | 3,200원 | 3,500원 | 4,000~6,000원 |
| ARPA (기업당 평균) | 27만 원 | 37만 원 | 50만 원+ |
| 평균 직원 수/고객사 | 25인 | 30인 | - |
| Churn Rate (월) | < 5% | < 3% | 2~5% |
| Net Revenue Retention | > 100% | > 110% | 110~130% |
| CAC | 50만 원 | 40만 원 | 30~80만 원 |
| LTV | 250만 원 | 400만 원 | 500만 원+ |

## 6. GTM (Go-to-Market) 전략

### 6.1 타겟 고객 세분화

| 세그먼트 | 기업 규모 | 산업 | 페르소나 | 핵심 페인포인트 | 타겟 플랜 |
|----------|----------|------|----------|----------------|-----------|
| **Primary** | 10~50인 | IT/스타트업 | HR 담당자 겸 경영지원 | 엑셀 HR 관리 한계, 법적 리스크 | Starter |
| **Secondary** | 51~100인 | 전문서비스, 커머스 | HR 팀 리드 | 수동 프로세스, 인사 평가 체계화 | Growth |
| **Tertiary** | 100~300인 | 제조, 유통 | CHRO/VP HR | 시스템 통합, 커스터마이징 | Enterprise |

### 6.2 채널 전략

| 채널 | 전략 | 예산 비중 | 기대 효과 |
|------|------|-----------|-----------|
| **콘텐츠 마케팅** | HR 관련 블로그, 노동법 가이드, 체크리스트 | 30% | 오가닉 리드 생성, SEO |

| **이벤트/웨비나** | HR 트렌드 웨비나, 스타트업 행사 후원 | 15% | 브랜드 인지도 + 리드 |
| **제품 주도 성장 (PLG)** | 무료 플랜 → 셀프서브 업그레이드 | 10% | 저비용 전환 |
| **아웃바운드 세일즈** | Growth/Enterprise 타겟 직접 영업 | 10% | 고가 계약 확보 |

### 6.3 세일즈 퍼널

```
인지 (Awareness)
  │  콘텐츠 마케팅, LinkedIn, 웨비나
  ▼
관심 (Interest)
  │  무료 체험 가입, 데모 신청
  ▼
평가 (Evaluation)
  │  14일 무료 체험 (Growth 기능 전체 개방)
  ▼
결정 (Decision)
  │  세일즈 콜, 온보딩 지원 제안
  ▼
구매 (Purchase)
  │  Starter or Growth 구독
  ▼
확장 (Expansion)
  │  직원 증가 → 자동 업셀, 부가기능 추가
  ▼
추천 (Advocacy)
     추천 프로그램, 케이스 스터디
```

**전환율 목표:**

| 퍼널 단계 | 전환율 목표 | 업계 벤치마크 |
|-----------|------------|--------------|
| 방문 → 가입 | 5~8% | 3~7% |
| 가입 → 무료체험 시작 | 40~50% | 30~50% |
| 무료체험 → 유료전환 | 15~25% | 10~20% |
| Starter → Growth 업그레이드 | 20~30% (12개월 내) | 15~25% |

## 8. 경쟁 차별화 전략

### 8.1 포지셔닝 매트릭스

