# Business Model: 명품 중고거래 플랫폼

## 1. Product Overview

### 제품 설명

명품 중고거래에 특화된 C2C 마켓플레이스. 일반 중고 플랫폼(당근마켓, 번개장터)이 커버하지 못하는 "명품 전문 감정 + 안전거래"를 핵심 가치로 제공한다.

### 핵심 가치

- **정품 인증**: AI 1차 스크리닝 + 전문가 2차 감정으로 가품 거래 차단
- **안전 거래**: 에스크로 결제로 구매자/판매자 모두 보호
- **전문 큐레이션**: 명품에 특화된 카테고리, 브랜드 가이드, 시세 정보 제공
- **낮은 수수료**: 크림 대비 2%p 이상 저렴한 수수료로 판매자 유인

### 타겟 고객

| 세그먼트 | 특성 | 니즈 |
|---------|------|------|
| **판매자 (Supply)** | 2030 여성, 명품 보유자, 트렌드 소비형 | 빠른 판매, 낮은 수수료, 간편한 리스팅 |
| **구매자 (Demand)** | 2030~40 남녀, 합리적 명품 소비 지향 | 정품 보장, 적정 가격, 다양한 상품 |
| **파워셀러** | 월 5건+ 거래, 개인 리셀러/보세 사업자 | 수수료 절감, 우선 노출, 판매 도구 |

### 제약 조건

- 월 운영 예산: 500만원 (인건비 별도)
- 개발 인력: 2명
- 투자 상태: 부트스트랩
- 지역: 한국 시장

## 3. Revenue Model

### 선택 모델: 하이브리드 (거래 수수료 + 선택적 구독)

거래 수수료를 핵심 수익원으로, 파워셀러 대상 프리미엄 구독을 보조 수익원으로 조합한다.

### 선택 근거

| 근거 | 설명 |
|------|------|
| **시장 검증** | 크림, The RealReal, Vestiaire 등 성공 사례 모두 거래 수수료 기반 |
| **고단가 적합** | 명품은 거래 단가 30~80만원 → 낮은 수수료율(7~8%)로도 건당 2~4만원 수익 |
| **진입 장벽 낮음** | 거래 시에만 비용 발생 → 사용자의 초기 등록 저항 최소화 |
| **팀 규모 적합** | 2명 팀으로 복잡한 구독 시스템보다 수수료 정산이 구현 용이 |
| **예측성 보완** | 구독 MRR로 거래 변동성 완화 (전체 매출의 5~10% 목표) |

### 기각 대안

| 모델 | 기각 이유 |
|------|----------|
| **순수 구독** | 명품 거래 빈도가 낮아(월 1~2건) 월정액 정당화 어려움. 전 세계적으로 순수 구독 명품 마켓 성공 사례 없음. 구독료를 높이면 가입자 수 급감, 낮추면 수익 불가. |
| **광고** | 초기 MAU 부족으로 광고 매출 비현실적. 명품 브랜드 광고는 MAU 10만+ 필요. 광고가 명품 UX를 훼손. |
| **종량제** | 마켓플레이스에 부적합. "사용량"의 정의가 모호하며, 거래 수수료와 본질적으로 동일. |
| **순수 수수료 (구독 없이)** | 작동하지만 매출 예측성이 낮고 파워셀러 리텐션 도구가 부재. 구독 레이어 추가로 MRR 안정성 확보 권장. |

### 가치 지표와 과금 단위

| 구분 | 정의 |
|------|------|
| **가치 지표** | "감정 인증된 안전한 명품 거래" — 고객이 돈을 내는 이유 |
| **과금 단위 (수수료)** | 거래 성사 금액의 % (판매자 부담) |
| **과금 단위 (구독)** | 월정액 고정비 — 수수료 할인 + 프리미엄 기능 번들 |

## 4. Pricing Strategy

### 가격 티어

#### Free (무료 회원)

| 항목 | 내용 |
|------|------|
| 가격 | 0원 |
| 거래 수수료 | **8%** (구매자 부담 없음, 판매자 부담) |
| 리스팅 | 월 5건 |
| 감정 | AI 기반 1차 감정 (사진+시리얼넘버) |
| 노출 순위 | 일반 |
| 정산 주기 | 거래 완료 후 7일 |

#### Premium Seller (프리미엄 셀러 구독)

| 항목 | 내용 |
|------|------|
| 가격 | **9,900원/월** |
| 거래 수수료 | **5%** (3%p 할인) |
| 리스팅 | 무제한 |
| 감정 | AI + 전문가 감정 우선 배정 |
| 노출 순위 | **우선 노출** (검색 상단, 홈 추천) |
| 정산 주기 | 거래 완료 후 **3일** (빠른 정산) |
| 추가 | 프리미엄 셀러 뱃지, 판매 통계 대시보드, 가격 추이 알림 |

#### 수수료 가격대별 차등

| 거래 금액 | Free 수수료 | Premium 수수료 |
|----------|-----------|---------------|
| ~30만원 | 8% | 5% |
| 30~100만원 | 7% | 4.5% |
| 100만원 이상 | 6% | 4% |

#### 부가 서비스 (단건 과금)

| 서비스 | 가격 | 설명 |
|--------|------|------|
| 전문가 실물 감정 | 15,000원/건 | 전문 감정사 실물 확인 + 감정서 발급 |
| 끌어올리기 | 2,000원/건 | 리스팅 상단 재노출 (24시간) |
| 프리미엄 리스팅 | 5,000원/건 | 홈 배너 영역 3일 노출 |
| 안심 위탁 보관 | 거래가의 3% | 플랫폼이 실물 보관 + 검수 + 배송 대행 |

### 무료/유료 경계 논리

- **무료 범위**: 리스팅 5건, AI 감정, 기본 검색/브라우징 → 명품 거래의 핵심 가치(감정+안전결제)를 경험 가능
- **유료 전환 동기**: 월 3건 이상 거래 시 구독 전환이 경제적 (3건 x 50만원 x 3%p 할인 = 45,000원 절감 vs 9,900원 구독료). "더 많이 팔수록 더 이득"
- **설계 원칙**: 가벼운 구매자/판매자는 무료로, 반복 거래하는 파워셀러는 구독으로 자연 전환

### 경쟁사 대비 포지션

```
저렴 ←―――――――――――――――――――→ 비쌈

당근(0%) ─── 번개장터(3.5~5%) ─── [우리(5~8%)] ─── 크림(10%) ─── 발란(15~25%)
                                       ↑
                              "크림급 전문성, 더 낮은 수수료"
```

**포지셔닝 전략**: 크림의 70~80% 가격에 동등한 감정 서비스를 제공하는 **가치 침투(value penetration)** 전략. "명품 전문성은 크림급, 수수료는 번개장터급"으로 양쪽 사용자 모두 유인.

## 6. Risk Assessment

### 상위 3 리스크

#### 리스크 1: 콜드 스타트 문제 (양면 시장 부트스트래핑)

| 항목 | 내용 |
|------|------|
| **발생 가능성** | 높음 (90%) |
| **영향도** | 치명적 — 판매자 없이는 구매자가 없고, 양쪽 모두 없으면 매출 제로 |
| **대응 방안** | (1) Supply-first 전략: 명품 커뮤니티에서 파워셀러 50~100명 직접 영입, 첫 3개월 수수료 0% (2) 시드 재고 20~30개 직접 매입 리스팅 (3) "핸드백" 단일 카테고리 집중 → 물량 밀도 확보 |
| **모니터링 지표** | 주간 신규 리스팅 수, 리스팅 대비 거래 전환율 |

#### 리스크 2: 가품 사고 / 신뢰 위기

| 항목 | 내용 |
|------|------|
| **발생 가능성** | 중간 (50%) |
| **영향도** | 치명적 — 1건의 가품 사고가 브랜드 전체 신뢰를 파괴 |
