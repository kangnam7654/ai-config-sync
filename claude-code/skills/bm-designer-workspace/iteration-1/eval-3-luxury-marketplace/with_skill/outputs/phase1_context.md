# Phase 1: 컨텍스트 수집 결과

## 수집 항목 정리

| 항목 | 내용 |
|------|------|
| **제품/서비스** | 명품 중고거래 플랫폼 (luxury secondhand marketplace) |
| **핵심 가치** | 명품 특화 — 감정/인증, 안전거래, 고급 UX |
| **타겟 고객** | B2C. 명품 중고 판매자(개인) + 구매자(개인). 2030 여성 비중 높음, 중상위 소득층 |
| **현재 수익 상태** | 없음 (pre-launch) |
| **경쟁 환경** | 일반 중고: 당근마켓, 번개장터 / 명품 특화: 크림(KREAM), 발란, 트렌비, 리셀링(MUST IT) |
| **차별점** | 명품 중고에 특화 (일반 중고 플랫폼 대비 전문성) |
| **비용 구조** | 월 예산 500만원, 개발자 2명 |
| **제약 조건** | 월 500만원 예산, 소규모 팀(2명), 부트스트랩 추정 |
| **투자 상태** | 부트스트랩 (언급 없으므로 가정) |
| **지역** | 한국 |
| **고민 사항** | 수수료 모델 vs 구독 모델 — 어떤 것이 적합한지 |

## 추가 추론 항목

- 명품 중고 플랫폼이므로 **신뢰/인증**이 핵심 가치
- 거래 단가가 높음 (평균 30~100만원 이상)
- 거래 빈도는 일반 중고 대비 낮음
- 가품 리스크가 사업 리스크의 핵심
