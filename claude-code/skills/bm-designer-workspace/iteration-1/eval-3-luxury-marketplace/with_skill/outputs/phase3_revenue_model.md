# Phase 3: 수익 모델 설계

## 선택 모델: 하이브리드 (거래 수수료 + 선택적 구독)

### 선택 이유

1. **거래 수수료가 핵심 엔진**: 명품 중고는 거래 단가가 높아(30~80만원) 낮은 수수료율로도 의미 있는 수익 창출 가능. 거래 발생 시에만 비용이 생기므로 사용자 진입 장벽이 낮음.
2. **구독은 파워셀러 부가 서비스**: 순수 구독으로 명품 플랫폼을 운영하는 사례가 전무. 단, 파워셀러(월 5건+)에게 수수료 할인 + 프리미엄 노출을 제공하는 선택적 구독은 안정적 MRR 확보에 기여.
3. **팀 규모에 적합**: 2명 개발팀으로 복잡한 구독 관리 시스템보다 거래 기반 수수료가 구현/운영 비용이 낮음.
4. **시장 검증 완료**: 크림, The RealReal, Vestiaire 모두 거래 수수료 기반. 시장이 인정한 모델.

### 기각 대안 및 사유

| 기각 모델 | 기각 이유 |
|----------|----------|
| **순수 구독** | 거래 빈도가 낮은 명품 시장에서 월정액은 이용자 이탈 유발. 구독료 설정이 어려움 (너무 높으면 가입 안 함, 너무 낮으면 수익 불가). 선례 없음. |
| **광고 모델** | 초기 트래픽이 부족한 신규 플랫폼에서 광고 매출은 비현실적. 명품 브랜드 광고는 최소 MAU 10만+ 필요. |
| **종량제** | 마켓플레이스에 부적합. API나 SaaS가 아닌 거래 플랫폼에서 "사용량"의 정의가 모호. |
| **순수 수수료 (구독 없이)** | 초기에는 이것만으로 충분하나, 매출 예측성이 낮고 파워셀러 리텐션 도구가 없음. 구독 레이어를 추가하면 MRR 안정성 확보. |

## 가치 지표와 과금 단위

| 구분 | 내용 |
|------|------|
| **가치 지표** | "안전하고 인증된 명품 거래" — 감정 인증 + 에스크로 결제 |
| **과금 단위 (수수료)** | 거래 성사 금액의 % (판매자 부담) |
| **과금 단위 (구독)** | 월정액 — 고정 비용으로 수수료 할인 + 부가 혜택 |

## 가격 티어 구성

### Tier 1: 무료 회원 (Free)

| 항목 | 내용 |
|------|------|
| 가격 | 0원 |
| 거래 수수료 | 판매 수수료 8% (구매자 부담 없음) |
| 리스팅 | 월 5건 |
| 감정 서비스 | 기본 AI 감정 (사진 기반) |
| 노출 | 일반 |
| 정산 주기 | 거래 완료 후 7일 |

### Tier 2: 프리미엄 셀러 (Premium Seller) - 구독

| 항목 | 내용 |
|------|------|
| 가격 | 9,900원/월 |
| 거래 수수료 | 판매 수수료 5% (3%p 할인) |
| 리스팅 | 무제한 |
| 감정 서비스 | AI 감정 + 전문가 감정 우선 배정 |
| 노출 | 우선 노출 (검색 상단, 홈 추천) |
| 정산 주기 | 거래 완료 후 3일 (빠른 정산) |
| 추가 혜택 | 프리미엄 셀러 뱃지, 판매 통계 대시보드 |

### 부가 서비스 (단건 과금)

| 서비스 | 가격 | 설명 |
|--------|------|------|
| 전문가 감정 | 15,000원/건 | 실물 감정. 감정서 발급 |
| 끌어올리기 | 2,000원/건 | 리스팅 상단 재노출 |
| 프리미엄 리스팅 | 5,000원/건 | 홈 배너 영역 3일 노출 |
| 안심 보관 (위탁) | 거래가의 3% | 플랫폼이 실물 보관 + 배송 대행 |

### 수수료 구조 (가격대별 차등)

| 거래 가격 | 기본 수수료 | 프리미엄 셀러 |
|----------|-----------|-------------|
| ~30만원 | 8% | 5% |
| 30~100만원 | 7% | 4.5% |
| 100만원~ | 6% | 4% |

경쟁사 대비 포지션: **크림(10%) 대비 저가 침투 전략**. "명품 전문성은 크림급, 수수료는 더 저렴하게"

## 초기 가격 범위 근거

- 크림 합산 수수료 ~10% 대비 8%로 진입 → 판매자 유인
- 번개장터 안전결제 3.5~5% 대비 약간 높지만, 명품 감정 서비스 포함으로 차별화
- 프리미엄 구독 9,900원 = 번개장터 플러스(4,900원)의 2배이나, 수수료 할인 폭이 더 커서 월 3건 이상 거래 시 손익분기
- The RealReal 구독 $5(약 6,500원) 대비 합리적 수준
