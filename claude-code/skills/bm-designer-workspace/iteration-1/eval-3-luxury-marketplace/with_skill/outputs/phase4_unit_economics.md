## 핵심 지표

### ARPU (월 유저당 평균 수익)

```
수수료 ARPU = AOV × 거래 건수 × Take Rate × (1 - PG수수료비중)
           = 50만 × 1.5 × 7.5% × (1 - 3%/7.5%)
           = 50만 × 1.5 × 7.5% - PG비용
           = 56,250원 - 22,500원(PG)
           ≈ 33,750원 (순수수료)

구독 ARPU (전환율 5%) = 9,900 × 5% = 495원/유저 평균

총 ARPU ≈ 34,245원/월
```

### LTV

```
LTV = ARPU × 평균 이용 기간
    = 34,245 × 10개월
    = 342,450원
```

### LTV/CAC

```
LTV/CAC = 342,450 ÷ 50,000 = 6.85:1
→ 건강 기준 3:1 이상 충족 (매우 건강)
```

### Payback Period

```
Payback = CAC ÷ 월 ARPU = 50,000 ÷ 34,245 = 1.46개월
→ 12개월 이내 기준 충족 (매우 빠름)
```

### Gross Margin

```
매출 = 거래 수수료 + 구독료
변동비 = PG 수수료 + 감정 비용(AI 기반이므로 낮음)

Gross Margin ≈ (수수료 수익 - PG 비용) / 수수료 수익
= (56,250 - 22,500) / 56,250 = 60%
→ 마켓플레이스 기준 40%+ 충족
```

### Break-even Point

```
월 고정비 = 500만원
건당 공헌이익 = AOV × (Take Rate - PG Rate) = 50만 × (7.5% - 3%) = 22,500원

BEP 거래 건수 = 5,000,000 ÷ 22,500 = 222건/월
→ 최소 월 222건 거래 필요 (약 148명의 활성 판매자, 유저당 1.5건)
```

## 민감도 분석

수익에 가장 큰 영향을 미치는 변수 3개: **AOV(거래 단가)**, **Take Rate(수수료율)**, **활성 유저 수**

### 기본 시나리오 12개월 기준 (월 419만원 매출)

| 변수 | -20% 변동 | 기본값 | +20% 변동 | 순이익 영향 |
|------|----------|--------|----------|-----------|
| **AOV** (50→40/60만원) | 월 매출 335만원 (-20%) | 419만원 | 월 매출 503만원 (+20%) | **가장 큰 영향** ±84만원 |
| **Take Rate** (7.5%→6%/9%) | 월 매출 335만원 (-20%) | 419만원 | 월 매출 503만원 (+20%) | ±84만원 (AOV와 동일 비중) |
| **활성 유저 수** (422→338/506명) | 월 매출 335만원 (-20%) | 419만원 | 월 매출 503만원 (+20%) | ±84만원 |

세 변수 모두 매출에 선형적으로 비례하므로 동일한 영향도를 가진다. 다만 **제어 가능성** 기준으로 우선순위를 매기면:

1. **활성 유저 수** — 마케팅/리텐션으로 직접 제어 가능. 가장 먼저 집중해야 할 지표.
2. **AOV** — 고가 명품 셀러 유치를 통해 간접 제어. 큐레이션 전략.
3. **Take Rate** — 경쟁 환경에 따라 조정. 초기에는 낮게 시작, 점진 인상.
