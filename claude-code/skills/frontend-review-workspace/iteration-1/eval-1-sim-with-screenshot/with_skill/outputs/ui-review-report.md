## Findings

### Finding 1: [Visual Hierarchy] 광고 배너가 핵심 콘텐츠보다 시각적으로 우위
- **위치**: 화면 상단 중앙 (약 y: 80~140px), 보라색 그라데이션 배너 "광고 보기 +5P"
- **현재**: 채도 높은 보라색(약 #9b59b6~#8e44ad) 배경의 CTA 버튼이 화면에서 가장 높은 시각적 가중치를 차지. 배너 위 "광고를 시청하고 5P를 받으세요" 텍스트까지 포함하면 화면 상단 약 20%를 광고가 점유.
- **문제**: 사용자가 앱을 열었을 때 가장 먼저 보는 것이 광고이며, 핵심 기능인 "오늘의 타로"와 "운세 기록"은 시각적 주목도가 낮음. Co-Star, Sanctuary 등 벤치마크 앱은 광고 없이 핵심 콘텐츠를 전면에 배치하거나, 광고를 하단/인라인으로 배치하여 UX 방해를 최소화.
- **개선**: 광고 배너를 운세 기록 리스트 사이(인라인 네이티브 광고)로 이동하거나, 시각적 가중치를 낮춰 배경과 유사한 톤(배경색 #1a1a2e 대비 약간 밝은 #2a2a3e, 텍스트 #a0a0c0)으로 변경. "오늘의 타로" 섹션의 제목을 24px bold, 색상 #ffffff로 강조하여 시각적 계층 역전.
- **벤치마크**: Co-Star는 광고 없는 구독 모델로 콘텐츠 집중 UX 제공. Sanctuary는 프리미엄 기능을 자연스러운 업셀 카드로 처리.

### Finding 2: [Navigation & Affordance] 하단 탭 바의 가독성 및 터치 타겟 부족
- **위치**: 화면 최하단 (약 y: 830~870px), 4개 탭 아이콘
- **현재**: 탭 바 아이콘이 약 20x20px 수준으로 매우 작고, 라벨 텍스트("생명", "위로로", "마이" 등)가 약 10px 크기의 저대비 색상(약 #666~#888)으로 배경과 거의 구분 불가. 현재 활성 탭을 나타내는 시각적 지표(하이라이트, 색상 변화, 인디케이터)가 관찰되지 않음.
- **문제**: Apple HIG 권장 최소 터치 타겟 44x44px 미달. 활성 탭 미표시는 사용자의 현재 위치 인식을 저해. WCAG 2.1 AA 기준 대비 3:1 미달 추정(#888 on #1a1a2e = 약 2.8:1).
- **개선**: 아이콘 크기 24~28px로 확대, 터치 영역 최소 44x44px 확보. 라벨 텍스트 12px medium, 색상 비활성 #8a8aa0 / 활성 #c084fc(밝은 보라) 또는 #ffffff. 활성 탭에 상단 2px 바 인디케이터 또는 아이콘 fill 변경 적용. 탭 바 배경을 #0d0d1a로 약간 어둡게 하여 콘텐츠 영역과 분리.
- **벤치마크**: Co-Star는 미니멀하지만 명확한 활성 상태 표시(흰색 fill 아이콘). Sanctuary는 컬러 아이콘+텍스트 라벨로 직관적 네비게이션 제공.

### Finding 3: [Typography] 텍스트 크기 계층 및 가독성 부족
- **위치**: 전체 화면 — 특히 "오늘의 타로" 제목(y: ~160px), 운세 기록 카드 내부(y: 350~700px)
- **현재**: "오늘의 타로"와 "운세 기록" 섹션 제목이 유사한 크기(약 16~17px)와 무게로 표시되어 계층 구분이 약함. 카드 내부 날짜("2026.03.16")는 약 12px, 제목("상세 운세")은 약 14px, 설명("오늘의 상세 운세 분석")은 약 12px로 차이가 미미. 설명 텍스트의 색상이 약 #8888aa 수준으로 배경 대비 낮음.
- **문제**: Material Design 및 Apple HIG 모두 명확한 타이포그래피 계층(Title/Headline/Body/Caption)을 권장. 현재 계층이 Title 16px → Body 14px → Caption 12px로 각 단계 차이가 2px에 불과하여 스캐닝이 어려움. 2026년 트렌드는 대담한 타이포그래피로 감정 전달을 강조.
- **개선**: 섹션 제목 20px semibold #ffffff, 카드 제목 16px semibold #f0f0f0, 카드 날짜 12px regular #9090b0, 카드 설명 14px regular #c0c0d0 (대비율 4.5:1 이상 확보). Line-height 1.5배 적용. "오늘의 타로" 제목에 약간의 letter-spacing(0.5px) 추가로 시각적 격상.
- **벤치마크**: Co-Star는 극단적으로 명확한 타이포그래피 계층 — 큰 제목과 작은 본문의 대비가 강렬. The Pattern은 감성적 세리프 폰트로 차별화.

### Finding 4: [Trend Alignment] 시각적 세련미 및 현대적 디자인 요소 부재
- **위치**: 전체 화면 — 특히 "오늘의 타로" 카드 영역(y: 150~320px), 배경 전체
- **현재**: 다크 퍼플 배경에 반투명 카드 스타일은 운세 앱에 적합한 방향이나, 표현이 단조로움. 타로 카드 이미지가 작은 직사각형(약 60x80px)으로 시각적 임팩트 부족. 배경에 미세한 성운/별 텍스처가 있으나 해상도가 낮아 보임. "김민정" 사용자 이름 아래 "↑ 김민정" 텍스트가 의미 불명확.
- **문제**: 2026년 운세 앱 트렌드는 3D 타로 카드, 글로우 효과, 파티클 애니메이션, 몰입형 비주얼. Golden Thread Tarot의 정교한 카드 일러스트, Co-Star의 세련된 미니멀리즘에 비해 시각적 완성도 격차가 큼. 배경 텍스처의 품질이 프리미엄 느낌을 저해.
- **개선**: (1) 타로 카드 이미지를 최소 120x180px로 확대, 카드 주변에 은은한 글로우 효과(box-shadow: 0 0 20px rgba(147, 51, 234, 0.3)) 추가. (2) 배경을 고해상도 성운 그라데이션(#0f0a1e → #1a1040 → #0d1b2a)으로 교체. (3) "오늘의 타로" 섹션에 카드 뒤집기 인터랙션 암시 요소(미세한 빛 반짝임 애니메이션) 추가. (4) 각 운세 기록 카드의 아이콘(해/구름/달)에 미세한 글로우 적용.
- **벤치마크**: Golden Thread Tarot의 금색 라인아트 카드 일러스트가 타로 앱 시각적 기준점. Mystic Mondays는 선명한 컬러 일러스트로 시각적 매력 극대화.

### Finding 5: [Color & Contrast] 카드 내 텍스트 가독성 부족
- **위치**: 운세 기록 카드 내부 (y: 350~750px), 특히 설명 텍스트 영역
- **현재**: 카드 배경이 약 #1e1e3a~#252545 (반투명 다크 퍼플)이고, 설명 텍스트("오늘의 상세 운세 분석", "긍정적인 에너지가 가득한 하루!")가 약 #7777aa~#8888bb 수준의 중간 톤 보라색. 추정 대비율 약 2.5~3.0:1.
- **문제**: WCAG 2.1 AA 기준 본문 텍스트 최소 4.5:1 대비율 미달. 다크 모드 모범 사례(Material Design)에서 medium-emphasis 텍스트도 60% opacity(약 #999)를 권장하며 이보다 낮은 대비는 가독성을 심각히 저해. 특히 야간 사용이 많은 운세 앱에서 저대비 텍스트는 사용자 이탈 원인.
- **개선**: 카드 설명 텍스트 색상을 #b0b0d0 (대비율 약 4.8:1)으로 변경. 카드 제목은 #e8e8f0 (대비율 약 8.5:1). 날짜 텍스트는 #9898b8 (대비율 약 3.5:1, 대형 텍스트 기준 충족). 카드 배경을 약간 더 어둡게(#1a1a35)하면 텍스트 대비가 자연스럽게 향상.
- **벤치마크**: Spotify 다크 모드는 #b3b3b3 텍스트 on #121212 배경으로 깔끔한 가독성 확보 (대비율 약 7.2:1).

### Finding 6: [Spacing & Alignment] 운세 기록 카드 간 간격 부족으로 시각적 밀집
- **위치**: 운세 기록 리스트 영역 (y: 350~750px), 카드 간 갭
- **현재**: 운세 기록 카드 3개가 약 6~8px 간격으로 밀착 배치되어 있어, 개별 카드의 경계가 시각적으로 뭉쳐 보임. 카드 내부 상하 패딩은 약 12~14px로 적절하나, 카드 간 외부 간격이 너무 좁음.
- **문제**: 8px 그리드 시스템 기준, 카드 컴포넌트 간 최소 권장 간격은 12~16px. 현재 간격은 카드들이 하나의 긴 블록처럼 보이게 하여 개별 항목의 독립성을 약화시킴. 사용자가 특정 기록을 빠르게 스캔하기 어려움.
- **개선**: 카드 간 간격을 12px로 확대. "오늘의 타로" 섹션과 "운세 기록" 섹션 사이에 24px 간격 적용. 섹션 제목("운세 기록") 상단에 32px 여백 추가. 각 카드에 1px border(rgba(255,255,255,0.06)) 또는 미세한 배경색 차이로 경계 강조.
- **벤치마크**: Sanctuary는 카드 간 16px 간격 + 라운드 코너(16px border-radius)로 각 카드가 독립적인 정보 단위로 명확히 인식됨.

### Finding 7: [Layout & Composition] "오늘의 타로" 섹션의 시각적 임팩트 부족
- **위치**: 화면 중앙 상단 (y: 150~320px), "오늘의 타로" 카드 섹션
- **현재**: 앱의 핵심 기능인 타로 카드 리딩이 약 170px 높이의 카드 안에 작은 카드 이미지(3장, 각 약 40x60px)로 표현됨. 카드 아래 "↑ 김민정" 텍스트의 의미가 모호. 전체 섹션이 운세 기록 리스트와 비슷한 시각적 비중으로 핵심 기능의 위상이 약함.
- **문제**: 타로/운세 앱의 핵심 가치인 "오늘의 운세/타로" 기능이 충분한 시각적 공간과 매력을 갖지 못함. 경쟁 앱들은 메인 화면의 50% 이상을 핵심 리딩에 할애하며 풍부한 비주얼로 사용자의 호기심과 참여를 유도.
- **개선**: "오늘의 타로" 섹션 높이를 최소 240px로 확대. 타로 카드 이미지를 80x120px 이상으로 키우고 카드 간 겹침(stacking) 레이아웃 적용. 카드 뒤집기/탭 유도 CTA("탭하여 카드 뒤집기") 추가. 섹션 배경에 미세한 그라데이션 또는 글로우 효과로 시각적 격상.
- **벤치마크**: Golden Thread Tarot는 화면 중앙에 대형 카드 이미지를 배치하고 탭하면 뒤집히는 인터랙션 제공. Mystic Mondays는 화려한 카드 일러스트가 화면을 지배하여 강한 시각적 끌림 생성.

## Quick Wins (< 1 hour each)
1. **카드 내 텍스트 색상 개선**: 설명 텍스트 #7777aa → #b0b0d0, 제목 텍스트 → #e8e8f0, 날짜 → #9898b8 로 변경하여 WCAG AA 대비율 충족
2. **하단 탭 바 개선**: 아이콘 크기 20px → 26px, 라벨 색상 비활성 #8a8aa0 / 활성 #c084fc, 활성 탭 인디케이터(상단 2px 바, color: #c084fc) 추가
3. **카드 간 간격 확대**: 운세 기록 카드 간 gap 6px → 12px, 섹션 간 gap → 24px 적용

## Strategic Improvements (require design iteration)
1. **"오늘의 타로" 섹션 전면 리디자인**: 타로 카드 크기 확대, 카드 뒤집기 인터랙션, 글로우 효과 추가로 앱의 핵심 기능을 시각적으로 격상 — 사용자 참여도와 리텐션에 직접적 영향
2. **광고 배치 전략 재설계**: 상단 배너 → 인라인 네이티브 광고 또는 운세 결과 화면 내 리워드 광고로 전환, 메인 화면의 프리미엄 느낌 복원
3. **비주얼 아이덴티티 강화**: 고해상도 배경 그라데이션, 타로 카드 커스텀 일러스트, 아이콘 세트 통일(이모지 → 커스텀 라인/글로우 아이콘)으로 브랜드 차별화
