# Idea Forge 최종 결과: PetGuard AI (AI 기반 반려동물 건강 모니터링 서비스)

## Phase 1: 아이디어 검증 루프 요약

### 라운드 수: 2 / 최대 2

### 라운드 1 (Rebuttal)
- **CEO 제안**: AI 웨어러블 디바이스 + 앱 → 바이탈 데이터 실시간 수집 → 이상 징후 조기 감지 → 수의사 상담 연계
- **CSO 거절 사유**: 하드웨어 제조 진입장벽 (초기 투자 5~10배), 웨어러블 착용 수용도 미검증 (치명적 리스크), 경쟁사 대비 차별화 불분명
- **핵심 피벗 포인트**: 하드웨어 제거, 소프트웨어 우선 전략으로 전환 필요

### 라운드 2 (Accept)
- **CEO 제안**: 스마트폰 카메라/마이크 기반 AI 건강 분석 앱 → 만성질환(관절염/비만/심장) 특화 → B2B2C 수의사 네트워크 연계
- **CSO 판정**: Accept (5개 기준 중 4개 긍정)
  - 긍정: 시장 타당성, 실행 가능성, 리스크 수준, 재무적 잠재력
  - 부정: 경쟁 우위 (기존 경쟁사 존재하나 통합 포지션 비어 있음)
- **핵심 차별화**: 하드웨어 불필요 + 복수 만성질환 통합 AI + 수의사 연계 원스톱

### 주요 피벗
라운드 1→2 사이의 핵심 변경:
1. 웨어러블 디바이스 → 스마트폰 카메라/마이크 (하드웨어 리스크 제거, 초기 투자 90%+ 절감)
2. 범용 건강 모니터링 → 만성질환 3종 특화 (AI 정확도 및 차별화 강화)
3. B2C 단독 → B2B2C 제휴 (고객 획득 비용 절감)

### 최종 제안 요약
스마트폰 카메라 기반 AI 건강 분석 앱으로, 보호자가 촬영한 반려동물 영상/사진에서 관절염, 비만, 심장질환 등 만성질환의 초기 징후를 AI가 감지하고, 이상 감지 시 제휴 동물병원/수의사 원격 상담으로 즉시 연결하는 B2B2C 서비스. 한국 30~50대 건강관리 적극 보호자 세그먼트 타겟.

### 즉시 실행 (0~3개월)
1. MVP: 카메라 기반 피부/눈 AI 분석 최소 기능 구현
2. 파일럿: 서울/경기 동물병원 10곳 제휴 계약
3. 데이터: AI 학습용 반려동물 건강 데이터 수집 체계 구축
4. 법률: 의료기기 규제 해당 여부 사전 확인

### 검증 필요 가설 (3~6개월)
- 무료→유료 전환율 4%+ (코호트 분석)
- 보호자 주 2회+ 촬영 습관 형성 (A/B 테스트)
- 월 ₩9,900~19,900 지불 의향 (베타 테스트)

### 마일스톤
- 3개월: MVP 출시, AI 정확도 80%+, 베타 500명
- 6개월: 무료 30,000명, 유료 전환 3%+, 제휴 병원 50곳
