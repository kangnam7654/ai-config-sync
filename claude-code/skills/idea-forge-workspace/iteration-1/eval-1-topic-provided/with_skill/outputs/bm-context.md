# BM Designer 외부 컨텍스트

## 제품 정보
- 제품/서비스 설명: 스마트폰 카메라/마이크 기반 AI 건강 분석 앱. 보호자가 촬영한 반려동물 영상/사진에서 행동 패턴 변화, 보행 이상(관절염 징후), 호흡 패턴, 피부/눈 상태를 AI가 분석하여 건강 이상을 조기 감지. 이상 감지 시 제휴 동물병원/수의사 원격 상담으로 즉시 연결.
- 핵심 가치 제안: 하드웨어 불필요(스마트폰만으로 AI 건강 분석), 만성질환(관절염/비만/심장) 특화 AI로 높은 정확도, 국내 동물병원/수의사 네트워크 B2B2C 연계
- 타겟 고객: 한국 30~50대 반려견/반려묘 보호자 중 동물병원 정기 방문자 (연 2회 이상, 건강관리 적극 세그먼트)
- 차별화 포인트: (1) 하드웨어 불필요 진입장벽 제거 (2) 복수 만성질환 통합 AI 분석 (기존 경쟁사는 단일 질환) (3) 수의사 네트워크 연동으로 분석에서 진료 원스톱

## 검증 이력
- CEO 제안 요약: 라운드 1에서 웨어러블 기반 모델 제안 후 CSO Rebuttal(하드웨어 리스크), 라운드 2에서 소프트웨어 우선+만성질환 특화+B2B2C로 피벗하여 CSO Accept
- CSO 검증 결과: Accept — 5개 기준 중 4개 긍정(시장 타당성, 실행 가능성, 리스크 수준, 재무적 잠재력). 경쟁 우위만 부정(기존 경쟁사 존재하나 통합 포지션 미점유)

## 리서치 데이터
- 시장 규모: TAM 한국 반려동물 시장 8.5조원(2024, 2032년 21조원 전망), SAM 헬스케어 1.5~2조원, SOM 디지털 헬스 500~1,000억원
- 경쟁사 분석: 티티케어(무료앱+KT부가서비스, 7개국 진출), 라이펫(무료앱+펫보험 커미션, 의료기기 허가), 핏펫(바이오키트+커머스), 펫닥(수의사 네트워크), PetPace(월 $12~25 구독)
- 업종 벤치마크: CAC $50~200(SaaS 평균), LTV/CAC 3:1 건강 기준, Churn 월 3.5%(B2B SaaS), ARPU 월 1~3만원 추정
- 가격 민감도 데이터: 월 양육비 19.4만원(전년 대비 4만원 증가), 연 의료비 102.7만원(2배 증가), 국내 경쟁사 대부분 무료+부가수익 모델, PetPace 글로벌 $12~25/월
