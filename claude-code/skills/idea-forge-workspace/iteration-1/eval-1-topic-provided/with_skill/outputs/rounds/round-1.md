# 라운드 1 브리핑

## 제안 요약

CEO는 AI 기반 웨어러블 디바이스 + 모바일 앱을 통해 반려동물의 바이탈 데이터(활동량, 수면 패턴, 심박수, 체온)를 실시간 수집하고, AI 모델이 이상 징후를 조기 감지하여 보호자에게 알림을 보내는 서비스를 제안했다. 타겟은 한국 30~50대 프리미엄 반려동물 보호자이며, 한국어 AI 건강 분석 엔진과 국내 수의사 네트워크 연동을 차별화 포인트로 제시했다. 수의사 원격 상담 연계 기능을 포함하여 단순 활동 추적기가 아닌 질병 조기 경고 시스템으로 포지셔닝했다.

## CSO 거절 사유

CSO는 5개 평가 기준 중 3개(실행 가능성, 리스크 수준, 경쟁 우위)가 부정이며, 치명적 리스크 1건(웨어러블 착용 수용도 미검증)을 식별하여 Rebuttal 판정을 내렸다.

구체적 약점:
1. **하드웨어 중심 모델의 높은 진입장벽**: 디바이스 설계, 제조, 인증, 재고 관리 등 초기 투자 부담이 소프트웨어 스타트업 대비 5~10배 이상. 한국 펫테크 스타트업 A사가 동물병원 연계 모델 실패로 폐업한 선례 존재.
2. **웨어러블 착용 수용도 치명적 불확실성**: 한국 보호자 대상 웨어러블 착용 의향 데이터가 없으며, 디바이스 착용 거부 시 서비스 자체가 성립 불가. 이는 해결되지 않으면 비즈니스 모델 전체를 무효화하는 치명적 리스크이다.
3. **차별화 불분명**: "한국어 AI + 국내 수의사 연동"만으로는 PetPace/Whistle 등 글로벌 플레이어 한국 진출 시 방어 불가. 핏펫/펫닥이 기존 고객 기반으로 웨어러블 추가 시 직접 경쟁 발생.

## Researcher 핵심 발견

1. **한국 반려동물 시장 8.5조원, 2032년 21조원 전망**: 시장 자체는 충분히 크고 성장 중이며, 보호자 최대 관심사 1위가 '건강관리'(55.2%)로 수요 기반은 탄탄하다.
2. **의료비 급증**: 반려동물 가구 평균 의료비가 2023년 57.7만원에서 최근 102.7만원으로 약 2배 증가, 건강관리 비용 지불 의향이 높아지고 있다.
3. **글로벌 경쟁사 이미 상용화 수준**: PetPace V3.0(월 $12~25), Whistle Health(10만+ 데이터 기반 AI) 등이 이미 클리니컬 그레이드 모니터링을 제공 중. 기술적으로 가능하나 데이터 확보가 핵심 병목.
4. **국내 플랫폼 경쟁**: 핏펫(헬스케어 솔루션+커머스), 펫닥(수의사 상담 네트워크), 에이아이포펫(AI 원격 건강검진)이 각자의 영역을 구축 중이나, 통합 웨어러블+AI+수의사 연계 서비스는 아직 부재.
5. **스타트업 실패 리스크 현존**: 반려동물 헬스케어 스타트업 폐업 사례(DailyVet 보도)에서 동물병원 연계 모델의 지속 가능성 문제 확인.

## 다음 라운드 제안 방향

1. **소프트웨어 우선 접근(Hardware-Light 전략)**: 웨어러블 자체 제조 대신, 스마트폰 카메라/마이크 기반 AI 분석(호흡 패턴, 행동 분석, 피부/눈 상태 이미지 분석)으로 출발하여 하드웨어 리스크를 제거하고, 검증 후 웨어러블로 확장하는 2단계 전략 검토.
2. **기존 플랫폼 제휴 모델**: 핏펫/펫닥 등 기존 고객 기반을 활용하는 B2B2C 모델로, 직접 고객 획득 비용을 낮추고 수의사 네트워크를 빠르게 확보하는 방향.
3. **니치 특화 전략**: 범용 건강 모니터링 대신 특정 만성질환(관절염, 심장질환, 비만)에 특화하여 AI 정확도를 높이고, 명확한 차별화를 확보하는 방향.

## 누적 이력

- **라운드 1**: 웨어러블 + AI + 수의사 연계 통합 서비스 제안 → 하드웨어 진입장벽, 착용 수용도 미검증, 차별화 불분명으로 CSO Rebuttal
