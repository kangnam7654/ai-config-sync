## 1. 아이디어 요약

스마트폰 카메라와 웨어러블 디바이스를 활용해 반려동물의 건강 상태를 AI로 실시간 모니터링하고, 질병 조기 발견, 맞춤형 건강 관리 가이드, 수의사 연결까지 원스톱으로 제공하는 서비스.

- 동물병원 1회 평균 진료비: **8만 3천 원**
- 가구당 2년간 평균 치료비: **78.7만 원** (월 3.3만 원)

### 2.4 반려동물 보험 시장

- 펫보험 가입률: **약 1.7% ~ 12.8%** (기준에 따라 상이)
- 2024년 수입보험료: 799억 원 (전년 대비 48.6% 증가)
- 비교: 스웨덴 40%, 영국 25%, 일본 20%
- 낮은 가입률 = 예방 건강관리에 대한 수요가 잠재적으로 큼

### 3.2 삼성전자의 진입 (2026.01)

CES 2026에서 삼성전자가 십일리터와 협업하여 스마트싱스 펫 케어 AI 건강 분석 기능을 공개. 치아 질환, 슬개골 탈구, 백내장, 비만도(BCS) 분석 등 4개 부문 서비스. 이는 대기업의 시장 진입으로 경쟁 강도가 높아졌음을 의미하지만, 동시에 시장 검증이 되었음을 보여줌.

### 3.3 경쟁 공백 (Whitespace)

현재 시장에서 **채워지지 않은 영역**:

1. **지속적 실시간 모니터링**: 기존 서비스는 대부분 "사진 1장 찍어서 분석" 방식. 웨어러블을 통한 24시간 연속 바이탈 모니터링(심박수, 체온, 활동량, 수면 패턴)과 AI 이상 탐지를 결합한 서비스는 부재
2. **예방 중심 건강 관리**: 질병 진단에 치중한 경쟁사들과 달리, 예방-모니터링-조기 발견-수의사 연결을 하나의 루프로 묶는 종합 서비스
3. **건강 데이터 기반 보험 연계**: 건강 데이터를 기반으로 한 동적 보험료 산정/할인 모델
4. **다견/다묘 가구 통합 관리**: 여러 반려동물을 한 대시보드에서 관리하는 기능

### 4.1 문제-솔루션 적합성 (Problem-Solution Fit)

| 검증 항목 | 판정 | 근거 |

| 타겟 고객이 충분한가? | **O** | 반려인구 1,546만 명, 시장 규모 8.5조 원 이상 |
| 기꺼이 비용을 지불하는가? | **O** | 월평균 19만 원 지출 중, 의료비 3.3만 원. 예방으로 비용 절감 가능성 인지 시 지불의사 존재 |
| 기존 대안보다 나은가? | **조건부 O** | 사진 기반 경쟁사 대비 "실시간 연속 모니터링"이 차별화 포인트. 단, 웨어러블 하드웨어 제작 비용과 사용자 접근성이 관건 |

### 4.2 SWOT 분석

**Strengths (강점)**
- 실시간 연속 모니터링은 현재 시장 공백
- AI 예측 모델을 통한 조기 경고는 높은 가치 제안
- 데이터 축적으로 시간이 갈수록 정확도 향상 (네트워크 효과)

**Weaknesses (약점)**
- 웨어러블 하드웨어 개발에 높은 초기 투자 필요
- 동물용 의료기기 인허가 절차 복잡 (3등급 의료기기)
- 반려동물이 디바이스 착용을 거부할 가능성

**Opportunities (기회)**
- 펫보험 가입률 1.7%로 성장 초기 단계, 보험사와 데이터 연계 기회
- 삼성의 스마트싱스 진입으로 시장 교육 비용 감소
- 고령 반려동물 증가 (2025년 반려동물 고령화 트렌드)
- 정부의 동물병원 진료비 공개 정책 (투명성 강화)

**Threats (위협)**
- 삼성 + 십일리터 협업으로 대기업 진입 본격화
- 기존 플레이어(티티케어, 핏펫)의 기능 확장 가능성
- 웨어러블 시장에서의 가격 경쟁
- 반려동물 데이터 관련 개인정보 우려

### 4.3 TAM-SAM-SOM

| 구분 | 정의 | 추정 규모 |
|------|------|----------|
| **TAM** (Total Addressable Market) | 한국 전체 반려동물 헬스케어 시장 | **1.5조 원** (8.5조 원 시장의 의료비 비중 약 18%) |
| **SAM** (Serviceable Available Market) | AI 기반 건강관리 서비스에 관심 있는 보호자 | **3,000억 원** (TAM의 20%, 기술 수용층) |
| **SOM** (Serviceable Obtainable Market) | 서비스 론칭 3년 내 확보 가능 시장 | **150억 ~ 300억 원** (SAM의 5~10%) |

### 4.4 종합 판정

**검증 결과: 조건부 GO**

- 시장 규모, 성장률, 고객 페인 포인트 모두 충분
- 경쟁 공백(실시간 연속 모니터링)이 존재하나, 삼성 등 대기업 진입이 위협
- **권장 전략**: 웨어러블 하드웨어 자체 개발보다는 **소프트웨어 우선(SW-first) 접근**, 기존 웨어러블 기기와의 연동 + 스마트폰 카메라 기반 분석으로 시작. 이후 데이터 축적 후 자체 하드웨어 출시 검토

## 5. 비즈니스 모델 설계

### 5.1 가치 제안 (Value Proposition)

> "반려동물의 건강을 24시간 지켜보는 AI 주치의"

**핵심 가치:**
1. **조기 발견**: AI가 건강 이상 징후를 조기에 감지하여 큰 병으로 악화되기 전에 알림
2. **비용 절감**: 예방 관리로 동물병원 진료비 절감 (평균 진료비 8.3만 원/회)
3. **안심**: 24시간 모니터링으로 보호자의 불안감 해소
4. **편의**: 동물병원 방문 전 상태 사전 파악, 수의사 원격 상담 연결

## 6. 진입 전략 (Go-to-Market)

### 6.1 Phase 1: MVP (0~6개월) - 소프트웨어 우선

- **핵심 기능**: 스마트폰 카메라 기반 AI 건강 체크 (눈, 피부, 치아, 체형)
- **차별화**: 사진 분석 + 증상 문진 AI 챗봇 조합으로 진단 정확도 향상
- **타겟**: 20~30대 여성 반려견 보호자 (얼리어답터)
- **채널**: 인스타그램, 반려동물 커뮤니티(강사모, 고사모 등)
- **목표**: MAU 10,000, 유료 구독 2,000명

### 6.2 Phase 2: 확장 (6~18개월) - 웨어러블 연동

- 기존 웨어러블 디바이스(펫피트 등)와 API 연동하여 활동량/수면 데이터 통합
- 수의사 원격 상담 기능 추가 (B2B 수의사 네트워크 구축)
- 반려묘 시장 확장
- 펫보험사와 제휴 (건강 데이터 기반 보험료 할인)

### 6.3 Phase 3: 플랫폼화 (18~36개월)

- 자체 웨어러블 디바이스 출시 검토 (ODM 방식으로 리스크 최소화)
- 사료/영양제 맞춤 추천 커머스 론칭
- 동물병원 EHR(전자건강기록) 연동
- 해외 진출 (일본, 동남아)

### 6.4 초기 마케팅 전략

1. **인플루언서 마케팅**: 반려동물 유튜버/인스타그래머 10명과 체험단 운영
