# ReviewMate (리뷰메이트) 사업 검증 리포트

> 작성일: 2026-03-20 | Idea Forge 파이프라인 산출물

---

## 한줄 요약

요식업 사장님의 리뷰 답변, SNS 마케팅, 재방문 유도를 AI로 자동화하는 SaaS. 시장성과 실현 가능성은 확인되었으나, **수익 구조(유닛 이코노믹스)와 경쟁 우위 확보가 사업화의 핵심 과제**입니다.

---

## 어떻게 이 아이디어를 찾았나? (Phase 0)

"요즘 뭐가 뜨는지 모르겠다"는 출발점에서, AI가 7개 유망 키워드를 자동 발굴하고 인기도/지속성을 정량 평가했습니다.

**최종 선정: "소상공인 AI 자동화"** (트렌드 점수 7.4/10)

선정 이유:
- AI 에이전트 시장이 연 175% 성장 중이지만, 대기업 영역이라 스타트업 진입이 어려움
- 반면 소상공인 AI 자동화는 경쟁 강도가 낮고, 정부가 디지털 전환 지원금을 대폭 확대 중
- 한국 소상공인 329만 사업체, 디지털 전환율 27.2%로 급증하며 수요가 실증됨

---

## 아이디어를 2번 갈아엎었습니다 (Phase 1)

### 첫 번째 시도: "소상공인 AI 올인원 SaaS"

AI가 POS 연동, 매출 분석, 고객 응대, 마케팅까지 다 해주는 "만능 서비스"를 제안했으나, 전략 검증에서 탈락했습니다.

**탈락 사유:**
- POS 제조사가 300개가 넘어서 기술적으로 통합이 거의 불가능
- 캐시노트가 이미 160만 사장님을 확보하고 있어, 같은 시장에서 후발주자로 승산이 낮음
- "모든 것을 다 한다"는 결국 "모든 것에서 2등"이 될 위험

### 두 번째 시도: "요식업 AI 마케팅 자동화 SaaS" (최종안)

첫 번째 실패를 반영해 3가지를 바꿨습니다:
1. **범위 축소:** 올인원 -> 마케팅 자동화에만 집중
2. **기술 리스크 제거:** POS 연동 포기 -> 카카오톡/네이버 API만 사용
3. **포지셔닝 변경:** 캐시노트와 경쟁 -> 캐시노트와 보완 관계

**5개 전략 기준 평가 결과: 4/5 통과** (시장성, 실현 가능성, 리스크, 재무성 긍정 / 경쟁 우위만 약함)

## 이 사업은 어떻게 돈을 버나? (Phase 2)

### 서비스 구성

| 기능 | 하는 일 | 가치 |
|------|--------|------|
| AI 리뷰 자동 답변 | 네이버 + 배달앱 리뷰에 AI가 답글 초안 자동 생성 | 하루 2시간 -> 10분 |
| AI SNS 콘텐츠 | 매장 사진/메뉴로 인스타/블로그 글 자동 작성 | 마케팅 비용 절감 |
| AI 재방문 유도 | 카카오톡으로 맞춤 쿠폰/메시지 자동 발송 | 재방문율 향상 |

### 가격표

| 플랜 | 월 가격 | 핵심 기능 |
|------|--------|----------|
| Free | 0원 | 리뷰 답변 초안 월 100건, 리뷰 대시보드 |
| Pro | 29,000원 | 자동 답변 게시 + SNS 콘텐츠 월 20개 + 리뷰 분석 |
| Premium | 49,000원 | Pro 전체 + 카카오톡 재방문 유도 + 상권 벤치마킹 |

참고: 마케팅 직원 1명 월급 200만원 vs 리뷰메이트 월 2.9만원

### 경쟁 환경

이미 비슷한 서비스가 있습니다:

| 경쟁사 | 특징 | 우리와 다른 점 |
|--------|------|---------------|
| 세일즈랩 | 배달앱 리뷰 AI + 정부 보조금 | 네이버/SNS 마케팅 없음 |
| 리본오토 | 네이버+구글+카카오 리뷰 통합 | 마케팅 콘텐츠/재방문 없음 |
| 딜리봇 | 배달앱 전문 리뷰 관리 | 오프라인 매장 미지원 |

**우리의 차별화:** 리뷰 + SNS + 재방문을 하나로 통합하고, 네이버 + 배달앱 + 카카오톡을 크로스 플랫폼으로 커버.

## 수익성 전망 (솔직한 평가)

### BM Score: 5.45 / 10 (C등급)

**양호한 부분:**
- 시장 매력도 (7/10): 요식업 소상공인 40만 타겟, 연 1,440억원 규모
- 수익 모델 (7/10): 프리미엄 모델이 소상공인에게 적합

**우려되는 부분:**
- 유닛 이코노믹스 (4/10): 고객 1명을 얻는 데 20만원, 벌어들이는 돈은 22만원. LTV/CAC 비율이 1.1:1로 건강 기준(3:1)에 크게 못 미침
- 경쟁 방어력 (4/10): 세일즈랩이 같은 방향으로 확장 중. 기능만으로는 차별화 부족

### 12개월 재무 전망 (기본 시나리오)

- 12개월 후 유료 고객: 약 375명
- 월 매출(MRR): 약 1,087만원
- 12개월 누적 적자: 약 6,100만원

## 가장 큰 리스크 3가지

1. **네이버가 AI 기능을 크게 강화하면?** -> 네이버가 못 하는 크로스 플랫폼(배달앱+카카오톡) 통합으로 방어
2. **세일즈랩이 같은 기능을 추가하면?** -> 카페 전문 AI처럼 특정 업종을 더 깊게 파서 정확도로 승부
3. **사장님들이 금방 해지하면?** -> 연간 계약 유도, 매장 성과 리포트로 "이 서비스 덕분에 매출이 올랐다"를 증명

## 최종 판단

**"가능성은 있지만, 바로 올인하기엔 이르다."**

시장은 존재하고 기술도 검증되었지만, 두 가지를 먼저 해결해야 합니다:
1. **유닛 이코노믹스 개선:** 현재 LTV/CAC 1.1:1은 돈을 벌기 어려운 구조. Churn을 줄이고 ARPU를 높여야 합니다.
2. **경쟁 방어력 확보:** 기능 번들링만으로는 세일즈랩에게 잡힐 수 있습니다. 특정 업종(카페) 특화 AI나 데이터 기반 상권 분석 같은 모방 어려운 해자가 필요합니다.

MVP를 빠르게 만들어서 실제 사장님 50명에게 써보게 한 뒤, 전환율과 Churn 데이터로 판단하는 것을 권장합니다.
