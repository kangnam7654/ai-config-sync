# Business Model: ReviewMate (리뷰메이트)

## 1. Product Overview

- **제품 설명:** 요식업 소상공인 특화 AI 마케팅 자동화 SaaS. 네이버 플레이스 + 배달앱(배민/요기요/쿠팡이츠) + 카카오톡 채널을 크로스 플랫폼으로 통합 관리하여, AI 리뷰 자동 답변, AI 마케팅 콘텐츠 자동 생성, AI 재방문 유도를 하나의 서비스로 제공한다.
- **핵심 가치:** 요식업 사장님의 마케팅 업무 시간을 하루 2시간에서 10분으로 줄인다.
- **타겟 고객:** 월매출 1,000만~5,000만원 규모의 요식업(카페, 식당, 베이커리) 소상공인. 네이버 플레이스에 등록되어 있고 배달앱도 운영하지만, 전담 마케팅 인력이 없는 1~3인 운영 매장.
- **차별화 포인트:** (1) 리뷰 답변 + SNS 콘텐츠 + 재방문 유도 3-in-1 통합, (2) 네이버 + 배달앱 + 카카오톡 크로스 플랫폼, (3) 캐시노트와 보완 관계 포지셔닝 (매출관리=캐시노트, 마케팅=리뷰메이트)

## 3. Revenue Model

### 선택 모델

| 항목 | 내용 |
|------|------|
| 수익 모델 | 프리미엄 (Freemium) + 구독 (Subscription) 하이브리드 |
| 선택 이유 | 소상공인은 가격 민감도가 높아 무료 티어로 제품 가치를 체험시킨 뒤 유료 전환이 필수. 경쟁사(세일즈랩, 리본오토)가 이미 무료 티어를 제공하므로 동일 전략 필요 |
| 가치 지표 | AI가 자동 처리하는 리뷰/콘텐츠/메시지 건수 |
| 과금 단위 | 월 구독 (정액제) — 소상공인은 비용 예측 가능성을 선호 |

## 4. Pricing Strategy

- **무료/유료 경계 논리:** Free 티어에서 리뷰 답변 초안을 경험하면 "자동 게시"와 "SNS 콘텐츠"에 대한 욕구가 생기도록 설계. 핵심 가치(시간 절감)를 충분히 체험하되, 자동화의 완성(자동 게시, 자동 발행)은 유료에서만 가능.
- **경쟁사 대비 포지션:** 세일즈랩(배달앱 중심)과 리본오토(리뷰 전용) 사이의 "크로스 플랫폼 마케팅 통합" 포지션. 가격은 동급이나 기능 번들이 더 넓음.
- **앵커링:** "마케팅 직원 1명 월급 200만원 vs 리뷰메이트 월 2.9만원" — 비용 대비 가치를 극대화하는 앵커.

## 5. Unit Economics

### 핵심 지표

| 지표 | 수치 | 근거 |
|------|------|------|
| CAC | 200,000원 | 정부 AI바우처/스마트상점 보조금 활용 시 일반 SMB CAC($300=39만원)의 50% 절감. 온라인 셀프서브 가입 병행 |
| ARPU (Blended) | 22,000원/월 | Free 0원 x 70% + Pro 29,000원 x 22% + Premium 49,000원 x 8% = 가중 평균 |
| 평균 이용 기간 | 10개월 | SMB SaaS 평균 6~12개월 (ARPU 낮은 제품 기준 보수적 추정) |
| LTV | 220,000원 | ARPU 22,000원 x 10개월 |
| LTV/CAC | 1.1:1 | 220,000 / 200,000 — 건강 기준(3:1) 대비 매우 낮음 |
| Payback Period | 9.1개월 | CAC 200,000 / 월 ARPU 22,000 |
| Gross Margin | 75% | SaaS 평균 70%+. AI API 비용(OpenAI/자체 모델)이 주요 변동비 |
| Contribution Margin | 35% | Gross Margin 75% - 고객 지원 인건비 등 직접 고정비 40% |

**주의: LTV/CAC 1.1:1은 건강 기준(3:1)에 크게 미달.** 개선 필요 항목:
1. Churn 감소 (평균 이용 기간 10개월 -> 18개월)로 LTV를 396,000원으로 개선 -> LTV/CAC 2.0:1
2. ARPU 향상: Pro/Premium 비율 증가 또는 연간 결제 할인으로 ARPU 30,000원 달성 -> LTV 300,000원 -> LTV/CAC 1.5:1
3. CAC 절감: 정부 보조금 의존도를 높이고, 입소문 기반 바이럴 계수 달성

## 7. Risk Assessment
