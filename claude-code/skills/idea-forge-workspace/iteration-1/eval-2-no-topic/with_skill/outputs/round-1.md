# 라운드 1 브리핑

## 제안 요약

CEO는 소상공인 대상 AI 올인원 SaaS "가게메이트"를 제안했다. 카카오톡 기반 AI 챗봇 고객응대, POS 연동 매출 분석, 재고 소진 예측, SNS 자동 마케팅을 월 3만원대로 제공하여 소상공인의 반복 업무를 30% 이상 절감하는 것이 목표다. 타겟은 월매출 500만~5000만원 규모의 요식업/소매업/미용업 소상공인으로, "사업자등록번호 입력만으로 설정 완료"라는 극단적 간편함을 차별화 포인트로 내세웠다.

## CSO 거절 사유

- **POS 파편화 기술 리스크 (치명적)**: 한국 POS 제조사 300개 이상이 난립하며 통합 API 표준이 부재. "사업자등록번호만으로 POS 연동"이라는 핵심 가치 이행이 현실적으로 극히 어렵다.
- **캐시노트 선점 효과**: 160만 사장님이 이미 사용 중인 캐시노트가 AI 기능을 추가하면 후발주자의 생존 공간이 급격히 축소된다.
- **올인원 차별화 부재**: 매출관리(캐시노트), 검색/예약(네이버), 배달(배민), CRM(쉽톡) 각 영역의 강자가 기능을 확장하면 "모든 것에서 2등"이 될 위험이 높다.
- 5개 평가 기준 중 긍정 2개(시장 타당성, 재무적 잠재력), 부정 3개(실행 가능성, 리스크, 경쟁 우위)

## Researcher 핵심 발견

1. 한국 소상공인 사업체 329만개, 디지털 전환율 27.2%로 급증 중. 정부 AI바우처 지원 확대.
2. 캐시노트가 카카오톡 기반으로 160만 사용자 확보. 핵심 성공 요인은 "쉬움". 단, AI 기능은 아직 미약.
3. POS 제조사 300개+ 파편화. 키오스크/테이블오더 시장은 춘추전국시대로 진입장벽 낮음.
4. 중소기업 AI 챗봇 사례에서 반복 질문 70%+ 자동화 달성 확인. 기술은 성숙 단계.
5. SMB SaaS 벤치마크: CAC $300~900, LTV/CAC 4:1 타겟. 한국 소상공인 특화 데이터는 부족.

## 다음 라운드 제안 방향

1. **버티컬 집중 전략**: 올인원 대신 "요식업 AI 마케팅 자동화"로 좁힌다. 캐시노트가 약한 영역(마케팅 자동화, 리뷰 관리 AI)에 집중하여 차별화.
2. **POS 의존도 제거**: POS 연동 대신 카카오톡 채널 + 네이버 리뷰 데이터 기반으로 출발. 기술 리스크를 최소화하고 빠른 MVP 출시.
3. **AI 마케팅에 특화된 가치 제안**: 리뷰 자동 답변 + SNS 콘텐츠 자동 생성 + 고객 재방문 유도 메시지 자동화라는 명확한 3가지 AI 기능으로 가치 입증.
