# MCP Gateway — BM 설계 문서

## 1. 제품 개요

**MCP Gateway**: MCP 서버의 프로덕션 운영을 위한 거버넌스 레이어 B2B SaaS

기존 MCP 서버 앞에 게이트웨이로 배치하여 코드 변경 없이 다음 기능을 제공:
- 모든 도구 호출의 자동 감사 로그 + SOC2/HIPAA/GDPR 컴플라이언스 매핑
- RBAC(역할 기반 접근 제어) + SSO/OAuth 2.1 통합
- 에이전트 행위 시각적 증적(Visual Proof) 대시보드
- 실시간 이상 탐지 알림 + 비용 추적

### 2-1. 프라이싱 구조: 하이브리드 모델 (플랫폼 피 + 사용량 기반)

|------|----------|--------------|--------------|------|
| Starter | $499/월 | 50,000 호출 | $0.005/호출 | SOC2 준비 중인 B2B SaaS 스타트업 |

| 수익원 | 가격 | 설명 |
|--------|------|------|
| 컴플라이언스 패키지 | $500~$2,000/월 | SOC2/HIPAA/GDPR별 사전 구성 정책 템플릿 + 자동 증적 생성 |
| Visual Proof 프리미엄 | $300~$1,000/월 | 에이전트 행위 시각적 재구성 + 감사인용 리포트 자동 생성 |
| 프로페셔널 서비스 | $200~$400/시간 | 온보딩, 커스텀 정책 설정, 감사 준비 컨설팅 |

### 2-3. 가격 결정 근거

- 엔터프라이즈 보안/컴플라이언스 SaaS 업계 벤치마크: ACV $30K~$100K+
- 규제산업 특화 솔루션: 수평 경쟁사 대비 15~30% 프리미엄 가능 (Deloitte 리포트 기준)
- Vanta(컴플라이언스 자동화) 참고: 시리즈C 기준 ARR $100M+, 평균 ACV ~$15K~$50K
- 하이브리드 모델 채택: 2026년 SaaS 기업 43% → 61% 하이브리드 모델 전환 추세 (Chargebee)

## 4. 비즈니스 모델 캔버스

### 핵심 파트너
- 클라우드 인프라: AWS/GCP/Azure (게이트웨이 호스팅)
- 컴플라이언스 자동화: Vanta, Drata (통합 파트너)
- SI/컨설팅: Deloitte, Accenture (규제산업 채널)
- MCP 생태계: Anthropic, OpenAI (프로토콜 표준화 참여)

### 핵심 활동
- MCP 프로토콜 네이티브 프록시 엔진 개발/유지
- 컴플라이언스 정책 템플릿 업데이트 (규제 변경 추적)
- Visual Proof 엔진 개발/고도화
- 엔터프라이즈 영업 + 감사인 교육

### 핵심 자원
- MCP 프로토콜 전문 엔지니어 (3~5명)
- 컴플라이언스 도메인 전문가 (금융, 헬스케어 각 1명)
- SOC2 Type II 자체 인증
- 규제산업 고객 레퍼런스

### 가치 제안
- **규제산업 CISO/컴플라이언스 팀**: "MCP 도입해도 감사에 걸리지 않습니다" — 코드 변경 없이 게이트웨이 배치만으로 SOC2/HIPAA 증적 자동 생성
- **플랫폼 엔지니어링 팀**: "AI 에이전트가 뭘 하는지 보입니다" — Visual Proof 대시보드로 에이전트 행위 완전 가시성
- **경영진**: "AI 거버넌스 비용을 80% 절감합니다" — 수동 감사 로그 작성/검토 자동화

### 고객 관계
- 엔터프라이즈: 전담 CSM + 분기별 비즈니스 리뷰
- Business: 셀프서비스 + 이메일 서포트
- Starter: 셀프서비스 + 커뮤니티

### 채널
- 직접 영업 (설립자 주도 → 엔터프라이즈 AE)
- 채널 파트너 (SI, 컴플라이언스 컨설팅)
- 콘텐츠 마케팅 (MCP 거버넌스 가이드, 감사 준비 체크리스트)
- 컨퍼런스/밋업 (RSA, AWS re:Invent, AI 보안 행사)

### 비용 구조
- 인건비: 70% (엔지니어 8~12명, BD 2~3명, 컴플라이언스 전문가 2명)
- 클라우드 인프라: 15% (프록시 처리량 비례)
- 영업/마케팅: 10%
- 기타(법률, 회계, SOC2 인증 유지): 5%

### 수익 흐름
- 월정액 구독 (SaaS 기본료)
- 사용량 과금 (MCP 호출량 초과분)
- 컴플라이언스 패키지 애드온
- 프로페셔널 서비스
