# Result

## Request
Android 에뮬레이터에서 앱 테스트 요청

## Response
**Redirected to mobile-dev agent.**

이 에이전트(iOS Simulator)는 iOS Simulator 전용입니다. Android 에뮬레이터 제어는 범위 밖이며, **mobile-dev** 에이전트가 담당합니다.

- iOS Simulator 에이전트 범위: `xcrun simctl` 기반 iOS Simulator 제어 및 Maestro UI 자동화
- Android 에뮬레이터 제어: **mobile-dev** 에이전트로 리다이렉트
