# service-dev Phase 0: Ideate

## 시작 시 확인 사항 체크

사용자 요청을 분석하여 3가지 사전 확인 항목을 정리합니다.

| 항목 | 사용자 입력 | 확인 상태 |
|------|-----------|----------|
| 아이디어 소스 | idea-forge에서 '반려동물 건강 관리 SaaS'로 검증 완료, BM도 나옴 | **확인됨** (경로 A: idea-forge 사용) |
| 디자인 필요 여부 | "모바일 웹이라 디자인도 필요해" | **확인됨** (Phase 2 실행 예정) |
| 서비스 이름 | 미제공 | **확인 필요** |

---

## Phase 0 진행: 경로 A (idea-forge 산출물 추출)

idea-forge에서 검증 완료되었으므로 경로 A를 따릅니다. idea-forge 산출물에서 4가지 필수 항목을 추출하여 정리합니다.

> **참고**: idea-forge 산출물 파일이 제공되지 않았으므로, 사용자에게 산출물 위치를 확인하거나 직접 4가지 항목을 구술해주실 것을 요청합니다.

### 사용자에게 요청할 사항

1. **idea-forge 산출물 파일 위치**: idea-forge 스킬의 결과물이 저장된 경로를 알려주세요. 해당 파일에서 아래 4가지 항목을 추출하겠습니다.

2. **서비스 이름 결정**: 산출물 저장 경로(`docs/{service-name}/`)에 사용할 이름이 필요합니다.
   - 제안: `pet-health-saas`
   - 다른 이름을 원하시면 알려주세요.

3. **Phase 0 산출물 확인**: idea-forge 산출물에서 아래 4가지 항목을 추출한 뒤 사용자님께 제시할 예정입니다. 산출물 파일이 없다면, 아래 항목을 직접 말씀해주셔도 됩니다.

---

### Phase 0 산출물 (확인 대기)

idea-forge 산출물에서 추출하여 아래 형식으로 정리할 예정입니다:

```
서비스 개요: {idea-forge에서 검증된 한 문장 서비스 설명}
핵심 기능: {idea-forge에서 도출된 3~7개 핵심 기능 목록}
타겟 사용자: {idea-forge에서 정의된 구체적 페르소나}
BM 요약: {idea-forge에서 설계된 비즈니스 모델}
```

현재 사용자 발화에서 파악된 정보만으로 초안을 구성하면:

```
서비스 개요: 반려동물 건강을 체계적으로 관리할 수 있는 모바일 웹 SaaS
핵심 기능: (idea-forge 산출물에서 추출 필요)
타겟 사용자: (idea-forge 산출물에서 추출 필요)
BM 요약: (idea-forge 산출물에서 추출 필요 — "BM도 나왔고"라는 발화로 존재 확인됨)
```

## Phase 0 게이트 체크리스트

- [ ] idea-forge 산출물에서 4가지 항목 추출 완료
- [ ] 서비스 이름 확정 (제안: `pet-health-saas`)
- [ ] 사용자가 4가지 산출물을 확인하고 승인

**위 3가지가 모두 완료되면 Phase 1 (Plan)으로 진행합니다.**

## 전체 Phase 로드맵 (현재 위치 표시)

```
>>> Phase 0: Ideate    ← 현재 (산출물 확인 대기)
    Phase 1: Plan      → 아키텍처 설계 + 구현 계획
    Phase 2: Design    → Figma UI/UX (모바일 웹 — 실행 예정)
    Phase 3: Spec      → AI용 개발 스펙 문서
    Phase 4: Build     → 모듈별 [dev→review→refactor→test] 사이클
    Phase 5: Document  → 사람용 문서
```

## 사용자님께 드리는 질문

Phase 0을 완료하기 위해 아래 사항을 확인해주세요:

1. **idea-forge 산출물 파일 경로**를 알려주세요. 해당 파일에서 서비스 개요, 핵심 기능, 타겟 사용자, BM 요약을 추출하겠습니다. 파일이 없다면 4가지 항목을 직접 말씀해주셔도 됩니다.

2. **서비스 이름**을 `pet-health-saas`로 사용해도 괜찮으신가요? 산출물은 `docs/pet-health-saas/` 경로에 저장됩니다.

3. 4가지 항목을 정리해드린 후 **승인해주시면** Phase 1(아키텍처 설계)로 넘어갑니다.
