## 1. 조회 접근 방식

### 사용할 사이트/URL

| 우선순위 | 사이트 | URL | 방식 |
|---------|--------|-----|------|
| 1 | 코레일 공식 (SRT 포함) | `https://www.letskorail.com/` | WebFetch로 시간표 API 호출 |
| 2 | 코레일톡 모바일 API | `https://smart.letskorail.com/` | WebFetch로 열차 조회 API 직접 호출 |
| 3 | 네이버 검색 | `https://search.naver.com/search.naver?query=서울+강릉+KTX+시간표` | WebSearch 활용 |

### 구체적 조회 절차

1. **WebSearch**로 `"서울 강릉 KTX 시간표 2026년 3월 21일"` 검색
   - 네이버/구글 검색 결과에서 코레일 시간표 또는 기차 시간표 사이트 확인
2. **WebFetch**로 코레일 열차 조회 페이지 접근
   - URL: `https://www.letskorail.com/ebizprd/EbizPrdTicketPr21111_i1.do`
   - 파라미터: 출발역=서울, 도착역=강릉, 날짜=20260321, 열차종류=KTX
3. 코레일 사이트가 동적 렌더링 필요 시, 네이버 기차 시간표 위젯 결과 활용
   - 네이버는 `서울 강릉 기차` 검색 시 시간표 카드를 직접 제공함

### 한계점 (Dry-Run에서 실제 조회 불가 이유)

- 코레일 시간표는 **동적 웹페이지**로, 단순 WebFetch로는 데이터 추출이 어려울 수 있음
- 2026년 3월 시간표는 **아직 공개되지 않았을 가능성** 있음 (코레일은 보통 1~2개월 전에 시간표 확정)
- 정확한 시간표는 반드시 실시간 조회가 필요함

3. **전화 예매**: 코레일 고객센터 1544-7788 (자동 ARS)
