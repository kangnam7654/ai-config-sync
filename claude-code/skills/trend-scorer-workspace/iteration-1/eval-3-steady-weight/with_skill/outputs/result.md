## 분석 요약

스테디 위주 가중치(alpha=0.30, beta=0.70)를 적용하면, **장기 운영 이력과 거대한 설치 기반**을 보유한 GitHub Copilot이 1위를 차지한다. 이는 "꾸준한 관심"이라는 평가 기준에서 3.5년간 일관된 성장을 보인 Copilot의 이점이 극대화된 결과다.

만약 기본 가중치(alpha=0.55, beta=0.45)였다면 순위는 역전되었을 것이다:
- Claude Code: (9.00 x 0.55) + (4.55 x 0.45) = 6.998
- Cursor: (7.75 x 0.55) + (5.95 x 0.45) = 6.940
- GitHub Copilot: (5.50 x 0.55) + (8.00 x 0.45) = 6.625

이는 각 도구가 서로 다른 차원에서 강점을 갖고 있음을 보여준다:
- **GitHub Copilot** = 안정성과 규모의 제왕 (STB 9, VOL 9)
- **Cursor** = 성장과 안정의 균형 (TS/SS 모두 상위권)
- **Claude Code** = 단기 화제성의 절대 강자 (TS 9.00, 전 Trending 지표 만점급)
