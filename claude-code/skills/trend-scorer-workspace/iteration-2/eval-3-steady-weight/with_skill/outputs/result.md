## 주제별 상세 분석

### 1위: GitHub Copilot (Final: 6.73)

- **왜 높은가**: 스테디 위주(β=0.70) 가중치에서 3년 이상의 안정적 사용 기반과 압도적 절대 볼륨(4.7M 유료 구독, 20M+ 전체 사용자, 42% 시장 점유율)이 결정적으로 작용했다. Trending 지표에서는 3개 도구 중 가장 낮지만, Steady 점수(7.10)가 확연히 앞선다.
- **Trending 근거**:
  - SVGR 5 — 유료 구독자 75% YoY 성장으로 꾸준하나, Claude Code/Cursor 대비 "급상승" 모멘텀은 약함. 최근 1주 검색 급증보다는 안정적 증가 패턴. ([GitHub Copilot Statistics 2026](https://www.getpanto.ai/blog/github-copilot-statistics))
  - SBI 6 — 2026년 3월 학생 무료 플랜에서 프리미엄 모델 제거 발표가 2,874 downvotes + 1,000 comments를 기록하며 큰 반응을 일으켰으나, 대부분 부정적 버즈. "Overrated in 2026" 같은 역풍 기사도 존재. ([WinBuzzer](https://winbuzzer.com/2026/03/16/github-removes-premium-models-copilot-student-plan-xcxwbn/), [Ryz Labs](https://learn.ryzlabs.com/ai-coding-assistants/why-github-copilot-is-overrated-in-2026-a-contrarian-view))
  - NFI 7 — 3월 중 Copilot Wave 3 발표(3/9), 학생 플랜 변경(3/12), Copilot Metrics GA(2/27), Coding Agent Semantic Search(3/17) 등 5건 이상의 주요 기사. ([GitHub Blog](https://github.blog/changelog/2026-02-27-copilot-metrics-is-now-generally-available/), [뉴스탭](https://www.newstap.co.kr/news/articleView.html?idxno=320999))
- **Steady 근거**:
  - STB 8 — 2022년 출시 이후 3년 이상 꾸준한 성장세. 30% QoQ 구독 성장률을 2024년 내내 유지. 급등/급락 없는 안정적 상승 곡선. Fortune 100의 90%가 채택한 산업 표준 도구. ([WeareTenet](https://www.wearetenet.com/blog/github-copilot-usage-data-statistics), [CompaniesHistory](https://www.companieshistory.com/github-copilot-statistics/))
  - VOL 9 — 20M+ 전체 사용자, 4.7M 유료 구독자로 AI 코딩 도구 중 최대 사용자 기반. 42% 시장 점유율. GitHub 플랫폼 자체의 1억+ 개발자 생태계와 직접 연결. ([Panto](https://www.getpanto.ai/blog/github-copilot-statistics))
  - SEA 3 — 엔터프라이즈 구매 사이클(분기별 예산 집행)과 일부 연동되나, 명확한 계절 패턴은 관찰되지 않음. 비계절적이나 분기별 소폭 변동 존재.
- **주의점**: "most loved" 설문에서 9%로 3개 도구 중 최저. 개발자 만족도와 시장 점유율 사이에 괴리가 있으며, 기존 GitHub 생태계 잠금(lock-in) 효과가 수치에 반영되었을 가능성. Trending 모멘텀이 약해 인기 위주(α=0.70) 가중치에서는 3위로 하락할 수 있다.

### 2위: Claude Code (Final: 6.22)

- **왜 높은가**: 2026년 3월 현재 Trending 지표(8.35)에서 3개 도구 중 압도적 1위. 일일 설치 수 급증, Reddit 최대 커뮤니티, 활발한 뉴스 사이클로 단기 모멘텀이 가장 강하다. 그러나 2025년 5월 출시로 역사가 10개월밖에 되지 않아 Steady 점수(5.30)가 낮고, 스테디 위주 가중치에서 불리하게 작용했다.
- **Trending 근거**:
  - SVGR 8 — 일일 설치 수(30일 이동평균)가 2026년 초 17.7M에서 29M으로 급증, 지수적 증가세 지속. $2.5B ARR 달성 속도가 ChatGPT보다 빠름. "ChatGPT moment"라는 표현이 복수 매체에서 사용됨. ([UncoverAlpha](https://www.uncoveralpha.com/p/anthropics-claude-code-is-having), [Heyuan](https://www.heyuan110.com/posts/ai/2026-02-23-agentic-coding-trends-2026/))
  - SBI 9 — r/ClaudeCode 주간 기여자 4,200+명으로 r/Codex(1,200명)의 3배 이상. 개발자 블라인드 테스트에서 67% 승률. Garry Tan의 Claude Code 설정 공유(3/12)가 TechCrunch 등에서 바이럴. ([DEV Community](https://dev.to/_46ea277e677b888e0cd13/claude-code-vs-codex-2026-what-500-reddit-developers-really-think-31pb), [TechCrunch](https://techcrunch.com/2026/03/17/why-garry-tans-claude-code-setup-has-gotten-so-much-love-and-hate/))
  - NFI 8 — 3월 중 Voice Mode 발표, /loop 명령어, Opus 4.6 1M 컨텍스트 GA, Garry Tan 논란, 쿼터 경제학 분석 등 다수 헤드라인. ([Releasebot](https://releasebot.io/updates/anthropic/claude-code), [Medium](https://medium.com/@william.couturier/claude-code-in-march-2026-the-economics-of-the-quota-792449b63edb))
- **Steady 근거**:

  - VOL 8 — claude.ai 월간 방문 176M(2025.12), 일일 설치 29M(2026.03), 기업 고객 300,000+. 절대 볼륨은 매우 크나 Copilot의 20M+ 유료 구독에는 못 미침. ([DemandSage](https://www.demandsage.com/claude-ai-statistics/), [Incremys](https://www.incremys.com/en/resources/blog/claude-statistics))
  - SEA 2 — 출시 10개월로 계절 패턴 판단이 불가능. 연간 반복 패턴이 형성되기엔 역사가 너무 짧음.
- **주의점**: 토큰 한도(쿼터) 불만이 지속적 이슈. "$20 플랜으로 12 프롬프트 후 소진"이라는 비판이 Reddit에서 반복됨. 가격 정책 변화가 장기 안정성에 영향을 줄 수 있다. 인기 위주(α=0.70) 가중치에서는 1위로 올라설 가능성이 높다.

### 3위: Cursor (Final: 6.17)

- **왜 높은가**: Trending(7.25)과 Steady(5.70) 모두 중간 수준으로 균형 잡힌 프로필. $2B ARR 달성, $50B 밸류에이션 협상, Graphite 인수 등 기업 성장 뉴스가 활발하나, Claude Code만큼의 단기 버즈는 없고, Copilot만큼의 장기 기반도 아직 없는 "전환기" 상태.
- **Trending 근거**:
  - SVGR 7 — 3개월 만에 ARR이 $1B에서 $2B로 두 배 증가(Bloomberg 3/2 보도). $50B 밸류에이션 협상(3/12)도 큰 관심 유발. 다만 Claude Code의 일일 설치 급증 대비 상대적으로 완만한 증가세. ([Bloomberg](https://www.bloomberg.com/news/articles/2026-03-02/cursor-recurring-revenue-doubles-in-three-months-to-2-billion), [TechCrunch](https://techcrunch.com/2026/03/02/cursor-has-reportedly-surpassed-2b-in-annualized-revenue/))
  - SBI 7 — Reddit에서 가장 많이 논의되는 AI 코드 에디터. Composer가 15개 파일에 걸친 React auth flow를 20분에 완성했다는 리뷰가 화제. 다만 프라이버시 우려와 가격 정책 변경(크레딧제 전환) 불만도 병존. ([AIToolDiscovery](https://www.aitooldiscovery.com/guides/cursor-reddit), [CheckThat.ai](https://checkthat.ai/brands/cursor/reviews))
  - NFI 8 — Bloomberg $2B ARR(3/2), $50B 밸류에이션(3/12), Graphite 인수, Composer 2 발표, JetBrains 통합, 자체 모델 개발 계획(3/19) 등 3월 한 달에 6건 이상의 주요 헤드라인. ([Bloomberg](https://www.bloomberg.com/news/articles/2026-03-12/ai-coding-startup-cursor-in-talks-for-about-50-billion-valuation), [DevOps.com](https://devops.com/cursor-acquires-graphite-to-streamline-ai-powered-development/))
- **Steady 근거**:
  - STB 6 — 약 2년의 역사(Copilot보다 짧고 Claude Code보다 길다). $200M → $500M → $1B → $2B ARR로 급격한 성장 곡선. 일관된 상승이지만, 2025년 6월 크레딧제 전환 시 CEO 사과 사태 등 변동성 요소 존재. 아직 완전한 안정기에 진입하지 않음. ([SaaStr](https://www.saastr.com/cursor-hit-1b-arr-in-17-months-the-fastest-b2b-to-scale-ever-and-its-not-even-close/), [Sacra](https://sacra.com/c/cursor/))
  - VOL 8 — 1M+ 일일 활성 사용자, 360K 유료 고객, Fortune 500의 50%+ 채택. Copilot의 절대 규모에는 못 미치나, 유료 전환율과 엔터프라이즈 침투율이 높음. ([Panto](https://www.getpanto.ai/blog/cursor-ai-statistics))
  - SEA 2 — AI 코딩 도구 전반에 계절성 패턴이 관찰되지 않음. Cursor도 마찬가지로 비계절적 도구.
- **주의점**: Anthropic, OpenAI와의 직접 경쟁 심화. 자체 모델 개발 계획(Bloomberg 3/19)은 양날의 검 — 차별화 가능성과 동시에 기존 모델 파트너십 리스크. 가격 정책 변동에 대한 개발자 불만이 잠재적 이탈 요인.

## 핵심 인사이트

1. **"꾸준함"의 승자는 Copilot**: 3년 이상의 역사, 42% 시장 점유율, 20M+ 사용자 기반은 단기 모멘텀으로 쉽게 뒤집히지 않는 안정성을 제공한다. 스테디 위주 평가에서 Copilot이 1위인 이유다.

2. **Claude Code는 "핫한 신성"**: Trending 지표 1위(8.35)지만 10개월 역사의 한계로 Steady가 낮다. 시간이 지나 안정화되면 종합 1위를 차지할 잠재력이 있으나, 쿼터 불만 해소가 선결 과제.

3. **Cursor는 "균형형"**: 어떤 가중치에서도 극단적 1위/3위가 아닌 중간 위치. 자체 모델 개발과 $50B 밸류에이션이 성사되면 판도가 바뀔 수 있는 변수.

4. **세 도구 모두 계절성이 낮다**: AI 코딩 도구는 아직 연간 반복 패턴이 형성되지 않은 카테고리. SEA 점수가 전반적으로 낮아 Steady 점수의 차별화는 STB와 VOL에서 결정된다.
