## 현재 화면 분석 (Saju 앱 메인 화면)

### 1. 전체 레이아웃 및 비주얼

**잘된 점:**
- 다크 테마 기반의 보라/남색 그라데이션 배경이 "운세/사주" 앱의 분위기에 잘 맞음
- 카드 기반 UI로 섹션이 명확하게 구분됨
- 하단 탭바가 있어 주요 기능 간 내비게이션이 명확함

**개선 필요:**
- 상단 광고 배너("광고를 시청하고 5P를 받으세요!")의 보라색 버튼이 배경과 너무 유사해 시각적 대비(contrast)가 부족함. CTA 버튼은 배경과 확실히 구분되는 색상 사용 권장
- 전체적으로 콘텐츠가 화면의 상단~중간에 집중되어 있고 하단에 여백이 많음

### 2. 타이포그래피

**잘된 점:**
- 섹션 제목("오늘의 타로", "운세 기록")의 크기와 굵기가 적절함
- 날짜(2026.03.16 등)와 본문 텍스트의 시각적 위계가 명확함

**개선 필요:**
- 상단 "달결" 타이틀 텍스트가 작아서 앱 브랜드 인지에 약간 부족함. 더 강조된 로고나 타이포 스타일 고려
- 광고 배너의 "광고를 시청하고 5P를 받으세요!" 텍스트가 배경 대비 가독성이 떨어짐

### 3. 카드 UI ("운세 기록" 섹션)

**잘된 점:**
- 각 카드에 날짜, 제목, 설명, 아이콘이 일관되게 배치됨
- 아이콘(해, 구름+해, 달)이 운세 유형을 직관적으로 구분해줌
- 카드 간 간격이 적절함

**개선 필요:**
- "전체 보기 >" 링크가 너무 작고 눈에 띄지 않음. 탭 가능 영역(tap target)이 44pt 이상인지 확인 필요
- 카드의 설명 텍스트("오늘의 상세 운세 분석", "긍정적인 에너지가 가득한 하루!")가 모두 한 줄로 처리되어 있어, 내용이 길어질 경우 truncation 처리 확인 필요

### 4. "오늘의 타로" 섹션

**잘된 점:**
- 카드 뒷면 이미지로 타로 테마를 잘 표현함

**개선 필요:**
- "1 장남함" 텍스트의 의미가 불분명함 (아마 "1 장 남음"의 오타로 보임). 띄어쓰기 및 문구 확인 필요
- 타로 카드 이미지가 작아서 탭 가능한 인터랙티브 요소인지 정적 디스플레이인지 구분이 어려움. 탭 유도 시각적 힌트(예: "탭하여 뽑기" 등) 추가 권장

### 5. 내비게이션 (하단 탭바)

**잘된 점:**
- 4개 탭으로 구성되어 과하지 않음
- 현재 선택된 탭(홈 아이콘)이 활성 상태로 구분됨

**개선 필요:**
- 탭 아이콘의 레이블이 스크린샷 해상도에서 잘 읽히지 않음. 충분한 크기인지 확인 필요
- 비활성 탭의 아이콘 색상이 배경과 너무 비슷해 대비가 부족함

### 6. 접근성 (Accessibility)

**개선 필요:**
- 전체적으로 다크 배경 위 밝은 텍스트의 대비비(contrast ratio)가 WCAG AA 기준(4.5:1)을 충족하는지 점검 필요. 특히 광고 배너 영역의 텍스트가 우려됨
- 광고 배너 버튼의 터치 영역이 충분한지 확인 필요 (최소 44x44pt)
- VoiceOver 등 스크린 리더에서 각 카드의 정보가 올바르게 읽히는지 확인 필요

### 7. 상태바

- 시간(1:34), 네트워크 아이콘, 배터리 등 시스템 상태바가 정상적으로 표시됨
- 밝은 색 상태바 텍스트가 다크 배경과 잘 어울림
