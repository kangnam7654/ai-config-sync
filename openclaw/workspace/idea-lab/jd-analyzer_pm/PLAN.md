# 주니어 서비스기획 JD 분석기 (옵션 B) — 계획

## 목표
- 주니어/신입 **서비스기획/PM** 지원자가 채용공고 텍스트를 붙여넣으면,
  - 요구역량/키워드를 구조화해서 보여주고
  - 자소서/포폴에서 무엇을 강조해야 하는지 체크리스트로 변환
  - 흔한 함정/미스매치 경고를 제공
- 수익: 광고(SEO 유입 기반)
- 운영: 고객응대 최소(셀프서브)

## MVP (1주)
### 입력
- 채용공고 텍스트(붙여넣기)
- 선택 옵션(토글/드롭다운): 경력(신입/주니어 고정), 도메인(커머스/콘텐츠/핀테크/플랫폼/기타)

### 출력(한 페이지)
1) 3줄 요약(역할/협업/성공지표 추정)
2) 키워드 맵(카테고리)
   - 문제정의/구조화
   - 데이터/지표
   - 실행/운영(프로세스)
   - 협업/커뮤니케이션
   - 문서화/산출물
   - 도메인 특화(커머스/콘텐츠 등)
3) 체크리스트 8~12개(포폴/자소서에 넣을 증거)
4) 함정 경고 3개(만능 JD, 데이터/실험 요구, 운영 비중 등)

## SEO 전략 (2주)
- 롱테일 랜딩: "서비스기획 채용공고 분석", "주니어 PM JD 키워드", "커머스 PM 자소서 키워드" 등
- 템플릿 기반으로 도메인별 가이드 페이지 10개부터

## 2주 판정 지표
- 세션 200+
- 분석 완료율 30%+
- D7 재방문 10%+

## 리스크/대응
- 경쟁(자소서 생성기)과 정면승부 피하기: 생성보다 "분해/체크/경고" 중심
- 신뢰: 과장된 '합격' 표현 금지, 참고용/가이드로 고지
