## Meme Translator / Meme Factory (Instagram automation)
- 목표: Gemini 제외, Instagram 업로드까지 end-to-end 테스트.
- Reddit 수집: API 키 발급이 막혀서 `www.reddit.com/.../top.json` JSON 엔드포인트(no-auth) 방식으로 진행.
  - `ProgrammerHumor`, `codingmemes` 대상으로 수집.
  - gallery/preview/i.redd.it(확장자 없음) 등에서 이미지 URL 추출 best-effort 로직 보강.
- 파이프라인(Editor→Designer→QC→Marketer)은 성공했고 `data/output_*.jpg` 및 캡션 메타(json) 생성됨.
- Instagram 업로드(비공식 instagrapi)에서 로그인 차단/실패:
  - HTTP 400 at `i.instagram.com/api/v1/accounts/login/` → instagrapi가 BadPassword로 매핑.
  - 비번은 앱에서 정상이나 자동화 로그인만 실패.
- 업로더 스크립트 개선:
  - 쉘 인자 깨짐 해결(os.system → subprocess list args)
  - uv 환경/의존성(instagrapi) 사용하도록 호출 개선(sys.executable)
  - `.env` 자동 로드(python-dotenv, find_dotenv(usecwd=True)) 추가
- 결론: 비용 0 + 완전 자동 + 안정성을 동시에 만족하기 어렵고, 업로드가 병목이라 프로젝트는 ‘폐기(코드는 보존)’ 결정.
- 프로젝트에 `POSTMORTEM.md`와 README 상태 표기 추가.
