## App Factory 착수 (app-factory)
- 목표: 사이클 기반으로 아이디어→타당성 게이트→스펙→Opencode 빌드→QA→리포트(텔레그램)까지 자동화.
- 레포: `~/projects/app-factory`
- Opencode 정책/스킬: `kangnam-opencode` (OpenClaw는 코드 작성 금지, Opencode가 계획/코딩 전담)
- Opencode agent 추천값: 구현/빌드 = **hephaestus**, 플래닝 = prometheus (고정 아님, 권장 기본값)
- 이슈: Opencode 실행이 간헐적으로 SIGKILL(signal 9)로 종료되는 불안정성.
- 관찰: PTY로 실행해야 로그가 잘 나오며, oh-my-opencode 규칙셋에서 write/edit/task 권한이 막히는 경우가 있어 `/start-work`로 권한 전환을 시도했음.
- 진행: `uv init --package app_factory`로 기본 uv 패키지 구조(`pyproject.toml`, `src/app_factory/`) 생성까지는 됨.
